I remove a link to another.

Begin micro-behaviour

REMOVE-LINK

Begin NetLogo code: 
substitute-text-area-for other-end-of-link the-other               
let other-end other-end-of-link
if link-with other-end != nobody [ask link-with other-end [die]]
End NetLogo code

Variants

You can specify the other end of the link by editing the text area.

Related Micro-behaviours

ADD-DIRECTED-LINK creates a directed link. ADD-UNDIRECTED-LINK creates an undirected link.

How this works

This uses the NetLogo link-with primitive with the value the-other to get the link and then removes it using the NetLogo die command.
