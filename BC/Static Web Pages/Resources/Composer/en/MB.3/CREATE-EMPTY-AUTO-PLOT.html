I set up an empty plot that others can add to.  It is scaled automatically as data is added.

Begin micro-behaviour

CREATE-EMPTY-AUTO-PLOT

Begin NetLogo code: 
substitute-text-area-for lower-left-corner-x 5    
substitute-text-area-for lower-left-corner-y 550 
substitute-text-area-for upper-right-corner-x 420  
substitute-text-area-for upper-right-corner-y 809 
substitute-text-area-for plot-label Hunger Plot     
substitute-text-area-for x-axis-label time        
substitute-text-area-for y-axis-label hunger        
substitute-text-area-for show-legends true  
create-plot
  "lower-left-corner-x" "lower-left-corner-y" ; lower left corner 
  "upper-right-corner-x" "upper-right-corner-y" ; upper right corner
  "plot-label" ; unique name (and label) for this plot
  "x-axis-label" ; X axis label
  "y-axis-label" ; Y axis label
   show-legends ; show legends when pens added (true or false)
End NetLogo code

Variants

You can change all the attributes in text areas to change the size, location, labels, minimum, or maximum values.

Related Micro-behaviours

This micro-behaviour is the best way to have multiple graphs on the same plot.  ADD-TO-PLOT can be used to add graphs.

This uses the NetLogo auto-plot feature to rescale the plot as needed. Use CREATE-EMPTY-PLOT if you wish to specify fixed minimum and maximum values. CREATE-AUTO-PLOT creates and draws an auto plot.

CREATE-HISTOGRAM creates histograms.
