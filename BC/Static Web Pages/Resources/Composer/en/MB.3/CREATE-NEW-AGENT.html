Required Micro-behaviours

This relies upon the variable the-initial-population which is defined by this slider.

Variants

This list of additional micro-behaviours can be changed.

Related Micro-behaviours

REPLACE-DEAD-AGENT uses this micro-behaviour. CREATE-NEW-AGENTS adds the INITIALISE-AGENT and SET-RANDOM-UNOCCUPIED-POSITION micro-behaviours. the-initial-population is defined by this slider.  CREATE-NEW-AGENTS-IN-LOWER-LEFT-CORNER replaces SET-RANDOM-UNOCCUPIED-POSITIONSET-RANDOM-UNOCCUPIED-POSITION-IN-LOWER-LEFT-CORNER.

How this works

It creates the-initial-population INITIALISE-AGENT and SET-RANDOM-UNOCCUPIED-POSITION. It subtracts one from the-initial-population since we include the original agent being copied in the population.
