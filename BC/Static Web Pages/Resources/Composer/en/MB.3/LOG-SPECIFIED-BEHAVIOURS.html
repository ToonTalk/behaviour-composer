I add to the log each time particular events happen to me.

Begin micro-behaviour

LOG-SPECIFIED-BEHAVIOURS

Begin NetLogo code: 
substitute-text-area-for behaviour-names "DIE" "KILL"                   
set log-behaviours [behaviour-names] ; quoted and separated by spaces
End NetLogo code

Any behaviour that is added or removed whose name contains DIE will cause an entry to be added in the log output area.

Variants

You can trace several behaviours at once. E.g.

set log-behaviours ["DIE" "EAT"]

Related Micro-behaviours

LOG-ALL-BEHAVIOURS logs all behaviours. You can also log attribute changes using LOG-ALL-ATTRIBUTES and LOG-SPECIFIED-ATTRIBUTES. SPECIFY-OUTPUT-AREA specifies up the dimensions and placement of the output area.
