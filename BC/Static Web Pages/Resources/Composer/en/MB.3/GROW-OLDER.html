I grow older.

Begin micro-behaviour

GROW-OLDER

Begin NetLogo code: 
substitute-text-area-for cycle-duration 1                      
          
do-every cycle-duration
   [set my-next-age time - my-birth]
End NetLogo code

After every "tick" this sets the object's value of age to the current simulation time minus the time of the object was created.

Variants

Any attribute can be repeatedly changed.

Related Micro-behaviours

GROW-HUNGRY also increases linearly with time. Age is implemented as the difference between now and birth while GROW-HUNGRY is implemented as a process that increases by one unit every second (by default).

How this works

AGE-IS-ZERO sets my-birth when I'm created and here I compute my-age by subtracting the current time from my birth time.
