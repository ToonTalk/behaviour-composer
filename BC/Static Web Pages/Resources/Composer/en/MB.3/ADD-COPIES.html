I create copies of myself at setup. The newly created copies can be given any additional behaviours.

Begin micro-behaviour

ADD-COPIES

Begin NetLogo code: 
substitute-text-area-for number-of-copies 9                                            
substitute-text-area-for comment Notes                                                 
add-copies 
    number-of-copies
    list-of-micro-behaviours "Additional behaviours" [SET-RANDOM-POSITION.html SET-RANDOM-HEADING.html]
;comment
End NetLogo code

Variants

You can replace the contents of the edit box by any positive integer or expression to make a different number of copies. You can change the list of behaviours that each newly created copy has. By default, there are just two behaviours: SET-RANDOM-POSITION and SET-RANDOM-HEADING. If you do change this behaviour we recommend you also change the name from ADD-COPIES to avoid confusion.

Related Micro-behaviours

ADD-COPY-OF-ANOTHER creates a copy of another individual. REPRODUCE is an example of creating copies under slightly plausible biological conditions.

History

ADD-COPIES was implemented by Ken Kahn. The text area for the number of copies was added in March 2009. The dynamic list of micro-behaviours was added in July 2009.
