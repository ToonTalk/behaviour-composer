I add to the log when an individual prefers to add a link.

Begin micro-behaviour

LOG-DESIRE-TO-ADD-A-LINK

Begin NetLogo code: 
substitute-text-area-for line-1 (word "Individual " who " desired to add a link to " [who] of my-other-player " at time " round time)
substitute-text-area-for line-2 (word "His utility of the network with this addition is " utility-of-network (list my-other-player) [])
substitute-text-area-for line-3 (word " which is greater than the utility of his network with no changes " utility-of-network [] [])
do-if my-desire-to-link = true
   [output-print line-1
    output-print line-2
    output-print line-3]
End NetLogo code

Variants

You can change the text of the message.

Related Micro-behaviours

This behaviour relies upon SIMPLE-ADD-LINK-DECISION to determine the value of my-desire-to-link. LOG-DESIRE-TO-REMOVE-A-LINK logs decisions to remove links.

How this works

This checks if I've set a variable indicating I prefer to add a link and if so adds a message to the log.
