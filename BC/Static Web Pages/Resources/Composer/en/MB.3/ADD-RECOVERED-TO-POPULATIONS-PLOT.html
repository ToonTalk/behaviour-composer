I add to the Populations Plot a plot of number of recovered individuals versus elapsed time.

Begin micro-behaviour

ADD-RECOVERED-TO-POPULATIONS-PLOT

Begin NetLogo code: 
substitute-text-area-for recovered-pen-color gray
substitute-text-area-for cycle-duration 1                      
         
do-after-setup
   [do-every cycle-duration
       [set-current-plot "Populations"
        create-temporary-plot-pen "Recovered"
        set-plot-pen-color recovered-pen-color        
        plotxy time 
               count all-individuals with [my-state = "recovered"]]]
End NetLogo code

Required Micro-behaviours

This expects CREATE-EMPTY-POPULATIONS-PLOT to be included to add the "Populations" plot.

Variants

You can edit the text box to have a different color.

Related Micro-behaviours

ADD-VACCINATED-TO-POPULATIONS-PLOT, ADD-INFECTED-TO-POPULATIONS-PLOT, ADD-SUSCEPTIBLE-TO-POPULATIONS-PLOT, and ADD-TOTAL-TO-POPULATIONS-PLOT can also add to the "Populations" plot.
