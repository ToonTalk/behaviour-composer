I add a new command to NetLogo to add a link to an object.

Begin micro-behaviour

ADD-LINK-TO

Begin NetLogo code: 
to add-link-to [node]
   ifElse is-list? my-next-player-links
     [set my-next-player-links fput node my-next-player-links]
     [set my-next-player-links (list node)]
   set my-next-player-links-set true
   ; create a NetLogo link:
   create-link-with node
end
End NetLogo code:

Related Commands

REMOVE-LINK-TO removes links. ADD-A-LINK-IF-IN-AGREEMENT uses this.

Related Micro-behaviours

ADD-NETWORK-FORMATION-PROCEDURES adds this reporter to the program.

How this works

If the player has any links the node is added. Otherwise a new list is created initially containing just the node being added.
