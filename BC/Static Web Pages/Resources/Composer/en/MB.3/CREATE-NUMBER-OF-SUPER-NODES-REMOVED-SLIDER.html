I create a slider controlling the number of super nodes (those with the most acquaintances) that should be removed from the model.

Begin micro-behaviour

CREATE-NUMBER-OF-SUPER-NODES-REMOVED-SLIDER

Begin NetLogo code: 
substitute-text-area-for slider-lower-left-corner-x 625 
substitute-text-area-for slider-lower-left-corner-y 10  
substitute-text-area-for slider-upper-right-corner-x 875 
substitute-text-area-for slider-upper-right-corner-y 50  
substitute-text-area-for slider-minimum-value 0   
substitute-text-area-for slider-maximum-value 100 
substitute-text-area-for slider-increment 1   
substitute-text-area-for slider-initial-value 5
create-slider 
  "the-number-of-super-nodes-removed" 
  "slider-lower-left-corner-x" "slider-lower-left-corner-y" ; lower left corner 
  "slider-upper-right-corner-x" "slider-upper-right-corner-y" ; upper right corner 
  "slider-minimum-value" "slider-maximum-value" ; minimum and maximum value
  "slider-increment" ; increment 
  "slider-initial-value" ; initial value
End NetLogo code

Variants

You can change the location, minimum, maximum, increment, and initial value.
