I add a message to the output area.

Begin micro-behaviour

ADD-MESSAGE

Begin NetLogo code:
substitute-text-area-for message Click on the GO button to run this model.         
output-print "message"
End NetLogo code

Variants

Edit the text of the message. You can add multiple lines by including multiple calls to output-print.

Related Behaviours

SPECIFY-OUTPUT-AREA specifies the dimensions and placement of the output area.

How this works

It calls the NetLogo output-print procedure to add text to the output area (by default at the bottom of the application).
