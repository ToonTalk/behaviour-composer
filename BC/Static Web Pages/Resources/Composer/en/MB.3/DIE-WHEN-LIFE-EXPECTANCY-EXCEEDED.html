When I exceed my life expectancy I die.

Begin micro-behaviour

DIE-WHEN-LIFE-EXPECTANCY-EXCEEDED

Begin NetLogo code: 
when time > my-life-expectancy
     [set dead true]
End NetLogo code

Variants

An alternative is to use DEATH-RATE which ignores my-life-expectancy and models the odds of dying each week.

Related Micro-behaviours

SET-MY-LIFE-EXPECTANCY sets my-life-expectancy.

How this works

When the time exceed my-life-expectancy I set a dead to true so that at the end of this time period, I'll be removed from the simulation.
