I change the initial size in pixels of each patch.

Begin micro-behaviour

SET-PATCH-SIZE

Begin NetLogo code: 
substitute-text-area-for initial-patch-size 12.5  
set-patch-size initial-patch-size
End NetLogo code

Variants

set-patch-size expects one number that is the number of pixels per patch. If, for example, the world is 10x20 patches then a patch size of 12.5 will use 125x250 pixels to display the world.

Related Micro-behaviours

SET-WORLD-SIZE defines how many patches are in the world.

How this works

Due to current limitations of NetLogo this specifies only the initial size of each patch and cannot be used to change the patch size during execution.
