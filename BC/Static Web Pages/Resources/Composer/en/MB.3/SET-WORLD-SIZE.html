I change the initial size of the world.

Begin micro-behaviour

SET-WORLD-SIZE

Begin NetLogo code: 
substitute-text-area-for minimum-world-x -49 
substitute-text-area-for maximum-world-x 50 
substitute-text-area-for minimum-world-y -49 
substitute-text-area-for maximum-world-y 50 
set-world-size minimum-world-x maximum-world-x minimum-world-y maximum-world-y
End NetLogo code

Variants

set-world-size expects 4 numbers: minimum x, maximum x, minimum y, and maximum y. For example a 100x100 grid could be created with values -49 to 50 inclusive (0 is included).

Related Micro-behaviours

SET-PATCH-SIZE defines how big each patch of the world is.

How this works

Due to current limitations of NetLogo this specifies only the initial size of the world and cannot be used to change the world dimensions during execution.
