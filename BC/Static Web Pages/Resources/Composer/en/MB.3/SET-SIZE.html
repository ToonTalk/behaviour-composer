I change my size.

Begin micro-behaviour

SET-SIZE

Begin NetLogo code: 
substitute-text-area-for size-of-prototype 2.0                       
set size size-of-prototype
End NetLogo code

Variants

You can change the contents of the text area to any positive number. A number less than 1 will make it appear smaller than the default size.

You can set the size to a random value by replacing the code in the box with NetLogo code. For example,

random-number-between .5 1.5

Related Micro-behaviours

SIZE-IS-FUNCTION-OF-AGE computes a size as a function of the age. SET-COLOR and SET-SHAPE change other aspects of the appearance of objects.

History

SET-SIZE was implemented by Ken Kahn.
