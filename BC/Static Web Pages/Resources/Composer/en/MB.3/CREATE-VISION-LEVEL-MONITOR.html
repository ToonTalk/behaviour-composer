I create a monitor for the mean level of vision in the population.

Begin micro-behaviour

CREATE-VISION-LEVEL-MONITOR

Begin NetLogo code: 
create-monitor
  "Mean Vision Level" ; label of the monitor
  "5" "250" ; lower left corner
  "105" "300" ; upper right corner
  "mean [my-vision-level] of all-individuals " ; value being monitored
  "3" ; number of decimals
End NetLogo code

Variants

All the parameters can be changed.

Related Micro-behaviours

CREATE-MEAN-METABOLISM-MONITOR, CREATE-TIME-MONITOR and CREATE-GINI-COEFFICIENT-MONITOR monitor other aspects of the model.

How this works

This computes the mean of the value of my-vision-level for all agents in the model.
