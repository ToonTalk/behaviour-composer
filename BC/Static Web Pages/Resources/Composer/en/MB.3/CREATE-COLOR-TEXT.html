I add some text to the NetLogo interface.

Begin micro-behaviour

CREATE-COLOR-TEXT

Begin NetLogo code: 
substitute-text-area-for lower-left-corner-x 5    
substitute-text-area-for lower-left-corner-y 250 
substitute-text-area-for upper-right-corner-x 80   
substitute-text-area-for upper-right-corner-y 300 
substitute-text-area-for font-size 11 
substitute-text-area-for text-color-number 0  
substitute-text-area-for transparent-background true  
substitute-text-area-for text-initial-lines "line 1" "line 2" "final line"                                                                   
create-color-text
  "lower-left-corner-x" "lower-left-corner-y" ; lower left corner 
  "upper-right-corner-x" "upper-right-corner-y" ; upper right corner
  "font-size" ; font size
  "text-color-number" ; text color number
  "transparent-background" ; transparent background (true or false)
  text-initial-lines ; lines of text quoted and separated by spaces
End NetLogo code

Variants

All the parameters can be changed. You can enter as many lines of text as you wish. You can specify other font sizes or colours or make the background transparent.

Related Micro-behaviours

CREATE-PLOT, CREATE-SLIDER, CREATE-MONITOR, CREATE-CHOOSER and CREATE-LEFT-TURN-BUTTON also create interface elements.

How this works

This creates a NetLogo text area.

History

This was implemented by Ken Kahn on 19 December 2009 based upon the older CREATE-TEXT.
