I set up an area for other micro-behaviours to draw graphs. The location, size, minimum, and maximum values can all be specified.

Begin micro-behaviour

CREATE-EMPTY-PLOT

Begin NetLogo code: 
substitute-text-area-for upper-left-corner-x 5    
substitute-text-area-for upper-left-corner-y 550 
substitute-text-area-for lower-right-corner-x 420  
substitute-text-area-for lower-right-corner-y 809 
substitute-text-area-for columns: 77 plot-label Name of this Plot     
substitute-text-area-for columns: 77 x-axis-label x label        
substitute-text-area-for columns: 77 y-axis-label y label        
substitute-text-area-for columns: 77 minimum-x-value 0   
substitute-text-area-for columns: 77 maximum-x-value 100 
substitute-text-area-for columns: 77 minimum-y-value 0   
substitute-text-area-for columns: 77 maximum-y-value 200 
substitute-text-area-for columns: 77 show-legends true  
substitute-text-area-for columns: 80 rows: 5 comment ; Notes                                                                                      
create-plot
  "upper-left-corner-x" "upper-left-corner-y" ; upper left corner (from the upper left corner of the applet) 
  "lower-right-corner-x" "lower-right-corner-y" ; lower right corner
  "plot-label" ; unique name (and label) for this plot
  "x-axis-label" ; X axis label
  "y-axis-label" ; Y axis label
  "minimum-x-value" ; minimum x value
  "maximum-x-value" ; maximum x value
  "minimum-y-value"  ; minimum y value
  "maximum-y-value" ; maximum y value
  show-legends ; legends (alternating pen names and color names or false (no legend) or true (legend generated via ADD-TO-PLOT))
comment
End NetLogo code

Variants

You can change all the attributes in text areas to change the size, location, labels, minimum, or maximum values.

Related Micro-behaviours

This micro-behaviour is the best way to have multiple graphs on the same plot. ADD-TO-PLOT can be used to add graphs.

CREATE-PLOT will set up the plot and draw it as well.

If you want the minimum and maximum values to be calculated dynamically use CREATE-EMPTY-AUTO-PLOT instead.

CREATE-HISTOGRAM creates histograms.
