I record a movie of a model's execution.

Warning -- this does not compress the movie so the file will be very large. You probably want to compress the movie with some other tool subsequently. The movie is made in Apple's QuickTime format but can be converted to other formats.

This only works when the model is run from within NetLogo. Applets can't create movies.

Begin micro-behaviour

MAKE-A-MOVIE-OF-A-MODEL

Begin NetLogo code: 
substitute-text-area-for movie-file-name test-movie.mov                 
substitute-text-area-for movie-duration 50  
do-after-setup
   [create-movie "movie-file-name" movie-duration true]
End NetLogo code

Related Micro-behaviours

SET-FRAME-DURATION can be used to set the duration of each frame (in seconds).

Variants

You can change the file name of the movie ("test-movie.mov" here) and the number of frames to record (50 in this example). Changing true to false will cause the movie to include the entire interface.

How it works

This produces a QuickTime movie file named "test-movie.mov". The file will be created relative to the folder where the NLOGO file was downloaded unless an absolute file name is given.
