I set the group I'm in. And I set my color to the group's color.

Begin micro-behaviour

SET-MY-GROUP

Begin NetLogo code: 
if-else the-groups-assigned-randomly
   [set my-group random-integer-between 1 the-number-of-groups]
   [set my-group 1 + (position self sort all-individuals with [kind = "Player"] 
                      mod the-number-of-groups)]
; compute NetLogo color where 15 is red, 25 is orange, etc.
set color my-group * 10 + 5 + my-group / 10
End NetLogo code

Variants

Change the range or the distribution of group identifiers.

Related Micro-behaviours

TOTAL-COST uses my-group.

How this works

This sets the value of my-group for each player to 1, 2, or 3.
