I create a slider controlling the cost of having a direct link between members of the same group.

Begin micro-behaviour

DIRECT-LINK-SAME-GROUP-COST-SLIDER

Begin NetLogo code: 
create-slider 
  "the-same-group-direct-link-cost" 
  "625" "10" ; upper left corner (from the upper left corner of the applet) 
  "875" "50" ; lower right corner
  "0" "1" ; minimum and maximum value
  ".01" ; increment
  ".4" ; initial value
End NetLogo code

Variants

You can change the location, minimum and maximum values, increment, and initial value.

Related Micro-behaviours

TOTAL-COST uses the value of the-same-group-direct-link-cost. DIRECT-LINK-BENEFIT-SLIDER adds a slider for the benefit of a direct link if in different groups or groups aren't defined.
