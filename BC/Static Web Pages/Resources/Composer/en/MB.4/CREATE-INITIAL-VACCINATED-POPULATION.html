Related Micro-behaviours

CREATE-INITIAL-VACCINATED-COUNT-SLIDER controls the-initial-vaccinated-population. This micro-behaviour relies upon SET-STATE-VACCINATED to set the state of the copies to "vaccinated". POSSIBLY-LOSE-IMMUNITY probabilistically restores susceptibility. SET-RANDOM-POSITION is used to initialise the location.

CREATE-INITIAL-INFECTED-POPULATION and CREATE-INITIAL-SUSCEPTIBLE-POPULATION initialise other parts of the population.

How this works

This creates the-initial-vaccinated-population copies of me that also run the SET-STATE-VACCINATED, POSSIBLY-LOSE-IMMUNITY, and SET-RANDOM-POSITION micro-behaviour.
