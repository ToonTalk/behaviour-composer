I display my current opinion by the direction I am facing.

Begin micro-behaviour

SET-HEADING-TO-CURRENT-OPINION

Begin NetLogo code: 
do-after .001    
   [do-every 1
      [set my-next-heading 90 * my-opinion mod 360]]
End NetLogo code

Variants

You can use other visual parameters to watch the opinion change.

Related Micro-behaviours

SET-SIZE-TO-UNCERTAINTY updates my size to reflect my uncertainty. RELATIVE-AGREEMENT updates the opinions.

How this works

This delays by .001 seconds so that it runs after all the opinions have been updated and then sets the heading the extreme positions are facing left (opinion = -1) or right (opinion = 1).
