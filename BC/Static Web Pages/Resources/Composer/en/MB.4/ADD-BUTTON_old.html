I create a button with an associated action.

Begin micro-behaviour

ADD-BUTTON

Begin NetLogo code: 
substitute-text-area-for lower-left-corner-x 5   
substitute-text-area-for lower-left-corner-y 225  
substitute-text-area-for upper-right-corner-x 107   
substitute-text-area-for upper-right-corner-y 255 
substitute-text-area-for columns: 77 button-label Add Behaviours
substitute-text-area-for columns: 35 recipients all-individuals
substitute-text-area-for keyboard-shortcut A
substitute-text-area-for columns: 80 rows: 5 comment ; Notes                                                                                      
substitute-text-area-for columns: 77 repeat-forever false
add-button 
  'upper-left-corner-x' 'upper-left-corner-y' ; upper left corner (from the upper left corner of the applet) 
  'lower-right-corner-x' 'lower-right-corner-y' ; lower right corner
  'button-label' ; label (cannot contain single quotes)
  'add-behaviours-to recipients list-of-micro-behaviours "Behaviours added by button" []' ; action
  'keyboard-shortcut' ; keyboard shortcut
  'repeat-forever' ; if true then this button will repeat its action forever
comment
End NetLogo code

Variants

All the parameters can be changed. If you don't want a keyboard shortcut then remove the last line of code.

Related Micro-behaviours

CREATE-PLOT and CREATE-SLIDER also create an interface element.

How this works

This creates a NetLogo button. The last parameter, the action, is available to micro-behaviours such as DO-BUTTON-COMMAND as the value of the button-command variable.
