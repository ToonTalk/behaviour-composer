I place a set of agents in a rectangular grid.

Begin micro-behaviour

LAYOUT-GRID

Begin NetLogo code: 
substitute-text-area-for columns: 62 rows: 2 upper-left-x min-pxcor
substitute-text-area-for columns: 62 rows: 2 upper-left-y min-pycor
substitute-text-area-for columns: 62 rows: 2 grid-width world-width
substitute-text-area-for columns: 62 rows: 2 grid-height world-height
substitute-text-area-for columns: 78 rows: 3 the-agent-set all-individuals                
substitute-text-area-for columns: 80 rows: 5 start-comment ; Notes                                                                                    
substitute-text-area-for columns: 80 rows: 2 end-comment ; Notes                        <br>;
start-comment
layout-grid 
  the-agent-set
  upper-left-x  ; upper left x
  upper-left-y  ; upper left y
  grid-width  ; width
  grid-height  ; height
end-comment
End NetLogo code

Variants

To change the location of the grid of agents change the upper left x and y. To make a grid of a different size edit the width and height. If you want separate rectangles for different kinds of agents, e.g. "Sheep" then add with [kind = "Sheep"] after all-individuals.

Related Micro-behaviours

LAYOUT-CIRCLE lays out agents in a circle. LAYOUT-SPRING computes a layout treating the links as springs.

SET-RANDOM-POSITION sets an agent's heading to a random value. SET-RANDOM-UNOCCUPIED-POSITION also finds a random position but only if it is unoccupied.

How this works

This uses the Modelling4All NetLogo layout-grid command to place the agents in a rectangle.
