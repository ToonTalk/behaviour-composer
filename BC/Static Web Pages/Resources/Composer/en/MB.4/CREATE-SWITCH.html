This has been deprecated in favour of DEFINE-BOOLEAN-PARAMETER. I create a switch that can be on or off.

Begin micro-behaviour

CREATE-SWITCH

Begin NetLogo code: 
substitute-text-area-for switch-variable the-switch-name                    
substitute-text-area-for upper-left-corner-x 625    
substitute-text-area-for upper-left-corner-y 10 
substitute-text-area-for lower-right-corner-x 875    
substitute-text-area-for lower-right-corner-y 50
substitute-text-area-for switch-initial-state true   
create-switch 
  "switch-variable" ; name of global variable
  "upper-left-corner-x" "upper-left-corner-y" ; upper left corner (from the upper left corner of the applet) 
  "lower-right-corner-x" "lower-right-corner-y" ; lower right corner
  "switch-initial-state" ; true (on) or false (off)
End NetLogo code

Variants

All the parameters can be changed.

Related Micro-behaviours

CREATE-PLOT, CREATE-SLIDER, CREATE-CHOOSER, CREATE-MONITOR and CREATE-BUTTON also create interface elements.

How this works

This creates a NetLogo switch for a global variable. In this example the variable is named switch-name and it is initially on.
