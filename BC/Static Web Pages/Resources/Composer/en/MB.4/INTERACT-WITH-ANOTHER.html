I interact with anyone who meets my conditions.

Begin micro-behaviour

INTERACT-WITH-ANOTHER

Begin NetLogo code: 
substitute-text-area-for columns: 65 set-of-agents all-individuals with [true]                           
substitute-text-area-for columns: 80 rows: 5 comment ; Notes                                                                                      
if can-pick-one set-of-agents
   [add-behaviours-to self           list-of-micro-behaviours "My Additional Behaviours" []
    add-behaviours-to the-other      list-of-micro-behaviours "Other's Additional Behaviours" []]
comment
End NetLogo code

Variants

You can replace the condition by using other NetLogo reporters to define a different set of agents. You can add behaviours to the lists. If you need to remove a behaviour you can add a behaviour that removes behaviours. See REMOVE-BEHAVIOURS.

Related Micro-behaviours

This uses the Modelling4All NetLogo command can-pick-one.

How this works

I generate a set of agents that meet condition. If the set is not empty then one agent is chosen from the set at random and called the-other. Micro-behaviours are added and removed  from the-other and myself.  This uses the Modelling4All NetLogo command can-pick-one which sets the-other to refer to another agent.
