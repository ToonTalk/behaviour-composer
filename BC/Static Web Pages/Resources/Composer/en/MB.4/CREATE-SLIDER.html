This has been deprecated in favour of DEFINE-PARAMETER. I create a slider for a parameter.

Begin micro-behaviour

CREATE-SLIDER

Begin NetLogo code: 
substitute-text-area-for slider-variable-name the-parameter               
substitute-text-area-for slider-upper-left-corner-x 625 
substitute-text-area-for slider-upper-left-corner-y 10  
substitute-text-area-for slider-lower-right-corner-x 875 
substitute-text-area-for slider-lower-right-corner-y 50  
substitute-text-area-for slider-minimum-value 0   
substitute-text-area-for slider-maximum-value 100 
substitute-text-area-for slider-increment 1   
substitute-text-area-for slider-initial-value 50  
substitute-text-area-for slider-units                       
substitute-text-area-for slider-horizontal true                    
create-slider 
  "slider-variable-name" ; the name of the parameter (a global variable)
  "slider-upper-left-corner-x" "slider-upper-left-corner-y" ; upper left corner (from the upper left corner of the applet) 
  "slider-lower-right-corner-x" "slider-lower-right-corner-y" ; lower right corner 
  "slider-minimum-value" "slider-maximum-value" ; minimum and maximum value
  "slider-increment" ; increment 
  "slider-initial-value" ; initial value
  "slider-units" ; units
  "slider-horizontal" ; display horizontally? (true or false)
End NetLogo code

Variants

All the parameters can be changed. The Behaviour Composer can provide more support if the parameter name begins with "the-". If multiple sliders have the same location they are placed one under another.

Related Micro-behaviours

CREATE-PLOT, CREATE-LEFT-TURN-BUTTON, and CREATE-MONITOR also create interface elements.

How this works

This creates a NetLogo slider.
