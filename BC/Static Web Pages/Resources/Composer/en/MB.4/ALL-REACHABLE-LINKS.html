This computes all the links that reachable from an initial list of nodes.

Begin micro-behaviour

ALL-REACHABLE-LINKS

Begin NetLogo code: 
to-report all-links-reachable-from [nodes]
 report all-links-reachable-from-2 nodes [] []
end

to-report all-links-reachable-from-2 [nodes nodes-already-visited links-reached]
 if nodes = [] [report links-reached]
 let new-nodes-already-visited sentence nodes nodes-already-visited
 let new-nodes []
 forEach nodes
    [let from-node ?
     ask ? 
         [ask link-neighbors
              [if not member? self new-nodes-already-visited and 
                  not member? self new-nodes 
                  [set new-nodes fput self new-nodes]]
          ask my-links 
              [if not member? self links-reached 
                  [set links-reached lput self links-reached]]]]
 report all-links-reachable-from-2 new-nodes new-nodes-already-visited links-reached
end
End NetLogo code

How this works

This repeatedly searches for the nodes reachable from the initial list of nodes or the most recent nodes found accumulating their links into a list.
