I initialise my local social network with a normal distribution. A link from me to another does not require that there be a link back to me.

Begin micro-behaviour

INITIALISE-SOCIAL-NETWORK-NORMAL-DISTRIBUTION-ASYMMETRIC

Begin NetLogo code: 
substitute-text-area-for minimum-acquaintances-normal-asymmetric-network 1 
substitute-text-area-for maximum-acquaintances-normal-asymmetric-network 10  
substitute-text-area-for average-acquaintances-normal-asymmetric-network 5  
substitute-text-area-for standard-deviation-acquaintances-normal-asymmetric-network 2  

do-after-setup
   [set my-acquaintances 
        n-of minimum count all-individuals round within-range (random-normal average-acquaintances-normal-asymmetric-network standard-deviation-acquaintances-normal-asymmetric-network) minimum-acquaintances-normal-asymmetric-network maximum-acquaintances-normal-asymmetric-network 
             all-individuals]
End NetLogo code

Variants

You can change the 1 (minimum number of acquaintances), 10 (maximum number of acquaintances), 5 (average number), or 2 (one standard deviation) to other positive integer values.

Or you may wish to replace the "5" with a variable defined by a slider for easy experimentation. See CREATE-SLIDER.

Related Micro-behaviours

INITIALISE-SOCIAL-NETWORK-CONSTANT-ASYMMETRIC differs from this in that everyone has the same number of acquaintances. An example of the use of this micro-behaviour is CHANGE-OPINION-SOCIAL-NETWORK.

INITIALISE-SOCIAL-NETWORK-NORMAL-DISTRIBUTION-SYMMETRIC creates an entire network where if A relates to B then B also relates to A.

How this works

It randomly selects n objects as my-acquaintances. It computes n using a random number with a normal distribution. It truncates the random number to the range (1 to 10 in this example). round is needed to convert to an integer value.

It is careful to avoid creating more acquaintances than there are individuals by taking the minimum of n and the number of individuals. This code is run after setup has completed since it is typically during setup that copies are created.
