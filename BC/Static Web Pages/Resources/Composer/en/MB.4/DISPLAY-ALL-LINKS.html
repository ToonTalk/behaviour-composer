I draw a line to each node I'm linked to.

Begin micro-behaviour

DISPLAY-ALL-LINKS

Begin NetLogo code: 
do-every 1
   [do-if is-list? my-player-links
       [foreach my-player-links
                [draw-line self ? white]]]
End NetLogo code

Variants

You can change white to another colour.

Related Micro-behaviours

REMOVE-DISPLAY-OF-LINKS removes the display of links.
