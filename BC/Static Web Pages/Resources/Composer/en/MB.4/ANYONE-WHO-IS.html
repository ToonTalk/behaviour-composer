I interact with anyone who meets my conditions.

Begin micro-behaviour

ANYONE-WHO-IS

Begin NetLogo code: 
substitute-text-area-for condition true                                             
anyone-who-is condition
       [add-behaviours-to self list-of-micro-behaviours "My Additional Behaviours" []
        add-behaviours-to the-other list-of-micro-behaviours "Other's Additional Behaviours" []]
End NetLogo code

Variants

You can replace true with any condition. You can add behaviours (and REMOVE-BEHAVIOURS) to the lists.

Related Micro-behaviours

This uses the Modelling4All NetLogo command anyone-who-is which sets the-other to refer to another agent.

How this works

I select someone at random that satisfies the condition and if there is anyone adds and removes micro-behaviours from the other and myself.
