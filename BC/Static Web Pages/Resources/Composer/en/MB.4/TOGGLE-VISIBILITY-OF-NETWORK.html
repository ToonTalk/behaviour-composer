This makes my semantic network hidden if it is showing and shows it if it is hidden.

Begin micro-behaviour

TOGGLE-VISIBILITY-OF-NETWORK

Begin NetLogo code: 

set my-network-visibility my-network-visibility != true
ask turtle-set all-nodes-reachable-from my-list-of-anchor-points
    [set my-visibility not my-visibility
     set hidden? not hidden?]
ask link-set all-links-reachable-from my-list-of-anchor-points
    [set hidden? not hidden?]
End NetLogo code

Related Micro-behaviours

SHOW-MY-NETWORK displays my network. HIDE-MY-NETWORK hides it. TOGGLE-VISIBILITY-OF-NETWORK switches its visibility. HIDE-ALL-NETWORKS hides all nodes and links.
