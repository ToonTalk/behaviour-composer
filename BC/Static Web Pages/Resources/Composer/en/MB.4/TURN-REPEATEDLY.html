I repeatedly turn a number of degrees.

Begin micro-behaviour

TURN-REPEATEDLY

Begin NetLogo code: 
substitute-text-area-for rotation-angle 10                            
substitute-text-area-for cycle-duration 1                             
          
do-every cycle-duration
   [turn-right rotation-angle]
End NetLogo code

Variants

You can change the contents of the text areas to other values or the value of a parameter (see DEFINE-PARAMETER). Using a negative value for the rotation will turn left instead.

Related Micro-behaviours

TURN-RANDOMLY turns back and forth a random amount.

How this works

This schedules a turn of number of degrees in the second box to occur every number of seconds in the first box.

History

TURN-REPEATEDLY was implemented by Ken Kahn.
