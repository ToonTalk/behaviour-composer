I update a parameter (or a NetLogo global variable) once.

Begin micro-behaviour

UPDATE-PARAMETER

Begin NetLogo code: 
substitute-text-area-for columns: 76 parameter-name the-parameter               
substitute-text-area-for columns: 76 parameter-expression the-parameter + 1                                                                                                  
substitute-text-area-for columns: 80 rows: 5 comment ; Notes                                                                                      
set parameter-name 
    parameter-expression
comment
End NetLogo code

Variants

You can change the name and the expression for updating the parameter.

Related Micro-behaviours

DEFINE-PARAMETER defines parameters that this behaviour updates. UPDATE-PARAMETER-REPEATEDLY updates parameters after every interval. UPDATE-ATTRIBUTE updates attributes of agents.

How this works

UPDATE-PARAMETER uses the NetLogo SET primitive command to update the value of the parameter.
