I change the location of the display of the world.

Begin micro-behaviour

SET-WORLD-LOCATION

Begin NetLogo code: 
substitute-text-area-for upper-left-world-x 200 
substitute-text-area-for upper-left-world-y 10
substitute-text-area-for lower-right-world-x 650 
substitute-text-area-for lower-right-world-y 550
set-world-location upper-left-world-x upper-left-world-y lower-right-world-x lower-right-world-y
End NetLogo code

Variants

set-world-location expects 4 numbers: upper left corner x, lower left corner y, lower right corner x, and lower right corner y. This example creates a 100x100 grid since -49 to 50 inclusive is 100 integers including 0.

How this works

Due to current limitations of NetLogo this specifies only the initial location of the world display and cannot be used to change the world location during execution.
