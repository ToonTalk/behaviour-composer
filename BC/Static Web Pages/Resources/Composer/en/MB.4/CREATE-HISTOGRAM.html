I set up and maintain a histogram.

Begin micro-behaviour

CREATE-HISTOGRAM

Begin NetLogo code: 
substitute-text-area-for upper-left-corner-x 5    
substitute-text-area-for upper-left-corner-y 550 
substitute-text-area-for lower-right-corner-x 420  
substitute-text-area-for lower-right-corner-y 809 
substitute-text-area-for columns: 77 histogram-label Size Distribution
substitute-text-area-for columns: 77 x-axis-label Size
substitute-text-area-for columns: 77 y-axis-label Number of individuals
substitute-text-area-for columns: 78 rows: 3 x-axis-value all-individuals
substitute-text-area-for columns: 78 rows: 3 y-axis-value size
substitute-text-area-for columns: 77 minimum-x-value 0  
substitute-text-area-for columns: 77 maximum-x-value 5  
substitute-text-area-for columns: 77 minimum-y-value 0  
substitute-text-area-for columns: 77 maximum-y-value 10
substitute-text-area-for columns: 78 legends "label" "red"       
substitute-text-area-for columns: 80 rows: 5 comment ; Notes                                                                                      
create-histogram
  "upper-left-corner-x" "upper-left-corner-y" ; upper left corner (from the upper left corner of the applet) 
  "lower-right-corner-x" "lower-right-corner-y" ; lower right corner
  "histogram-label" ; unique name (and label) for this plot
  "x-axis-label" ; X axis label
  "y-axis-label" ; Y axis label
  (x-axis-value) ; the population
  (y-axis-value) ; value being displayed
  "minimum-x-value" ; minimum x value
  "maximum-x-value" ; maximum x value
  "minimum-y-value" ; minimum y value
  "maximum-y-value" ; maximum y value
  legends ; legends (alternating pen names and color names or false for no legends)
comment
End NetLogo code

This example creates a histogram of the size of all-individuals.

Variants

One can make histograms of any numeric values. This example maintains a histogram named "Size Distribution". You can restrict the population by replacing all-individuals with another expression such as all-individuals with [kind = "fish"].

You don't need to specify the location of the plot if the default is acceptable. Just remove those lines.

Note that currently the system is limited to interpreting calls to create-histogram where each argument is on a different line.
