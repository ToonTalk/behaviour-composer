I select two random individuals and add behaviours to them.

Begin micro-behaviour

SELECT-TWO-RANDOM-PLAYERS

Begin NetLogo code: 
do-every 1
   [let players sort all-individuals with [kind = "Player"]
    let total length players
    if (total < 2) [stop]
    let i random total
    let j random total
    while [i = j]
          [set j random total]
    let player-i item i players
    let player-j item j players
    ask player-i [set my-other-player player-j]
    ask player-j [set my-other-player player-i]
    add-behaviours-to player-i list-of-micro-behaviours "Additional behaviours of player i" 
                      []
    add-behaviours-to player-j list-of-micro-behaviours "Additional behaviours of player j" 
                      []]
End NetLogo code

Variants

Other micro-behaviours could be added to the newly paired players.

Related Micro-behaviours

This adds to each node in the pair the micro-behaviours SIMPLE-ADD-LINK-DECISION, SIMPLE-REMOVE-LINK-DECISION, ADD-A-LINK-IF-IN-AGREEMENT and PREPARE-FOR-NEXT-TURN.

How this works

This first finds all the players that are not linked (i.e., uninitialised? my-other-player) and calls them players. It then generates two random values between 0 and the number of unpaired individuals (i and j). It then links the player with index i and the player with index j. It then adds micro-behaviours to the newly paired players to make decisions about add and removing links.
