This implements the copying of an incoming node.

Begin micro-behaviour

COPY-BELIEF-NODE

Begin NetLogo code: 

to-report copy-belief-node [belief-node source]
 let node-copy nobody 
 ask belief-node
      [hatch 1
             [set my-visibility true
              set my-time-of-acquisition time
              set my-motivation-level my-maximum-motivation-level
              set my-repetitions []
              set my-source source
              set node-copy self
              if is-string? [label] of one-of objects with [kind = "Anchor Point"]
                 [set label ifelse-value is-string? my-name [word my-name "     "] [""]]]]
 report node-copy
end
End NetLogo code
