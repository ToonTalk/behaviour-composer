I compute the noise (perhaps due to sensory/motor errors) for what direction I move.

Begin micro-behaviour

DIRECTION-NOISE

Begin NetLogo code: 
substitute-text-area-for direction-noise-standard-deviation 0.1  
substitute-text-area-for cycle-duration 1                      
       
do-every cycle-duration       
   [set my-direction-noise
        random-normal 0.0 direction-noise-standard-deviation * radian]
End NetLogo code

Variants

Change the standard deviation from 0.1 radians. Or use a different random number distribution.  You can change the frequency the noise changes by replacing the 1.

Related Micro-behaviours

TURN-IN-DIRECTION-AT-MAXIMUM-SPEED uses the "direction-noise" computed here. INFORMED-NOISE models the uncertainty about which direction is preferred.

How this works

On every cycle this computes a new random angle. The mean is zero and the standard deviation is 0.1 radians.
