I create a button for speeding up.

Begin micro-behaviour

CREATE-SPEED-UP-BUTTON

Begin NetLogo code: 
substitute-text-area-for upper-left-corner-x 5   
substitute-text-area-for upper-left-corner-y 225  
substitute-text-area-for lower-right-corner-x 107   
substitute-text-area-for lower-right-corner-y 255  
substitute-text-area-for button-label SPEED UP    
substitute-text-area-for acceleration 0.1     
substitute-text-area-for keyboard-shortcut F
create-button 
  "upper-left-corner-x" "upper-left-corner-y" ; upper left corner (from the upper left corner of the applet) 
  "lower-right-corner-x" "lower-right-corner-y" ; lower right corner
  "button-label" ; label
  "set my-next-speed my-next-speed + acceleration" ; action
  "keyboard-shortcut" ; keyboard shortcut
End NetLogo code

This assumes a micro-behaviour such as GO-FORWARD-AT-SPEED is also used.

Variants

All the parameters can be changed. If you don't want a keyboard shortcut then remove the last line of code. Buttons can also be edited within NetLogo but those changes won't be saved.

Related Micro-behaviours

CREATE-SLOW-DOWN-BUTTON is for slowing down (i.e. undoing the effects of this behaviour). CREATE-RIGHT-TURN-BUTTON and CREATE-LEFT-TURN-BUTTON are for turning.

CREATE-PLOT and CREATE-SLIDER also create an interface element.

How this works

This creates a NetLogo button. The last parameter, the action, is available to micro-behaviours such as DO-BUTTON-COMMAND as the value of the button-command variable.
