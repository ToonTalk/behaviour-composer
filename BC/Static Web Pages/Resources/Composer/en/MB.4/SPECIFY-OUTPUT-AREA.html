I describe the placement of the area where messages are added. The default is a narrow area at the bottom of the model interface. You can remove the output area by giving it a zero width or height.

Begin micro-behaviour

SPECIFY-OUTPUT-AREA

Begin NetLogo code: 
substitute-text-area-for upper-left-corner-x 624   
substitute-text-area-for upper-left-corner-y 10   
substitute-text-area-for lower-right-corner-x 1246  
substitute-text-area-for lower-right-corner-y 546  
substitute-text-area-for font-size 12 
create-output-area    
  "upper-left-corner-x" "upper-left-corner-y"  ; upper left corner (from the upper left corner of the applet) 
  "lower-right-corner-x" "lower-right-corner-y"  ; lower right corner
  "font-size"  ; font size
End NetLogo code

Variants

You can change the position and dimensions of the output area and the font size.

Related Micro-behaviours

ADD-MESSAGE adds custom messages to the output area. LOG-ALL-BEHAVIOURS, LOG-SPECIFIED-BEHAVIOURS, LOG-ALL-ATTRIBUTES, and LOG-SPECIFIED-ATTRIBUTES add messages to the output area.

How this works

This defines the corners of the area where NetLogo textual output is placed.
