I create a monitor for a value or computation.

Begin micro-behaviour

CREATE-MONITOR

Begin NetLogo code: 
substitute-text-area-for monitor-label parameter      
substitute-text-area-for lower-left-corner-x 5    
substitute-text-area-for lower-left-corner-y 250 
substitute-text-area-for upper-right-corner-x 80   
substitute-text-area-for upper-right-corner-y 300 
substitute-text-area-for monitor-value random-number-between 0 1
substitute-text-area-for monitor-number-of-decimals 4 
create-monitor 
  "monitor-label" ; label of the monitor
  "lower-left-corner-x" "lower-left-corner-y" ; lower left corner 
  "upper-right-corner-x" "upper-right-corner-y" ; upper right corner
  "monitor-value" ; value being monitored
  "monitor-number-of-decimals" ; number of decimals
End NetLogo code

Variants

All the parameters can be changed. It can also be edited within NetLogo but those changes won't be saved.

Related Micro-behaviours

CREATE-PLOT, CREATE-SLIDER and CREATE-LEFT-TURN-BUTTON also create interface elements.

How this works

This creates a NetLogo monitor that continually displays the value. In this example a random number between 0 and 1.
