I set my state to infected.

Begin micro-behaviour

SET-STATE-INFECTED

Begin NetLogo code: 
substitute-text-area-for infected-index-color yellow   
substitute-text-area-for infected-color red   
set my-next-state "infected"
ifElse time < .001 [set color infected-index-color] [set color infected-color]
add-behaviours-to my-last-encounter list-of-micro-behaviours "Additional Behaviours When Infected" [INCREMENT-TRANSMISSION-COUNT.html]
End NetLogo code

Related Micro-behaviours

SET-STATE-SUSCEPTIBLE, SET-STATE-VACCINATED, and SET-STATE-RECOVERED also set my-state. INCREMENT-TRANSMISSION-COUNT adds to my count of transmitted infections. If you want to model recovery then add RECOVERY-POSSIBLE-AFTER-INFECTION which will run after this sets my-state to "infected".

How this works

This creates the variable my-state and sets it to "infected". It sets my colour to red. It also keeps track of how many infections were transmitted from the same individual (stored in the my-last-encounter variable). If the time is less than 1, in other words this is an initial condition, then the index color is set to yellow.
