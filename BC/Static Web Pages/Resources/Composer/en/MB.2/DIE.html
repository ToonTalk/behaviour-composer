I die, i.e., I am removed from the simulation.

Begin micro-behaviour

DIE

Begin NetLogo code: 
set dead true
End NetLogo code

Variants

If you want an agent to remain (perhaps for generating statistics) but to leave the model then use HIDE instead. Note that, unlike this micro-behaviour, HIDE will not recycle the computational resources used.

Related Micro-behaviours

KILL uses DIE on another agent.

How this works

At the end of every simulation step all agents marked as dead are removed from the simulation.
