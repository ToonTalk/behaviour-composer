I add behaviours if a random number is greater than the given odds.

Begin micro-behaviour

DO-WITH-PROBABILITY

Begin NetLogo code: 
substitute-text-area-for odds 0.5                         
do-with-probability odds
   [add-behaviours list-of-micro-behaviours "Behaviours" []]
End NetLogo code

Variants

You can edit the box containing the odds. You can add micro-behaviours to the list by clicking on the micro-behaviour and navigating back to this page.

Related Micro-behaviours

ADD-BEHAVIOURS -- I add behaviours immediately (same as this micro-behaviour if the odds are equal to 1)

REPEATEDLY-DO-BEHAVIOURS -- Adds behaviours repeatedly.

DO-AFTER -- Delays the addition of a list of micro-behaviours.

WHEN -- Adds and removes micro-behaviours the first time that conditions are true.

WHENEVER -- Adds and removes micro-behaviours every time conditions are true.

ADD-BEHAVIOURS-TO-LINKS -- Adds micro-behaviours to links (rather than agents).

How this works

This relies upon the do-with-probability Modelling4All NetLogo command that produces a uniform random number between 0 and 1 and if it is greater than or equal to odds adds the specified micro-behaviours.

History

DO-WITH-PROBABILITY was first implemented by Ken Kahn on 6 October 2009.
