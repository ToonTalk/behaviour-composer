I remove a link if I decided to do so earlier.

Begin micro-behaviour

REMOVE-A-LINK-IF-DECIDED

Begin NetLogo code: 
do-after .002
   [do-if my-decision-to-unlink = true and link-neighbor? my-other-player
       [ask link-with my-other-player [die]
        ask my-other-player [if not uninitialised? my-next-player-links [set my-next-player-links remove myself my-next-player-links]]
        set my-accumulated-remove-link-utility-gains 
            my-accumulated-remove-link-utility-gains + my-remove-link-utility-gain
        set my-remove-links-count my-remove-links-count + 1
        set my-decision-to-unlink false]]
End NetLogo code

Related Micro-behaviours

SIMPLE-REMOVE-LINK-DECISION sets my-decision-to-unlink.

How this works

If I have decided to remove a link then I remove the other player from my list of links and have the other player remove me from its list.
