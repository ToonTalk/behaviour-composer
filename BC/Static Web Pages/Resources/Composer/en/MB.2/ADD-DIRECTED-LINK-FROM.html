I create an directed link from another to me.

Begin micro-behaviour

ADD-DIRECTED-LINK-FROM

Begin NetLogo code: 
substitute-text-area-for other-end-of-link the-other               
create-link-from other-end-of-link
End NetLogo code

Variants

You can specify the other end of the link by editing the text area.

Related Micro-behaviours

ADD-UNDIRECTED-LINK creates an undirected link. REMOVE-LINK removes a link.

How this works

This uses the NetLogo create-link-from primitive with the agent returned by the-other.

History
