I initialise an attribute used as my maximum life expectancy.

Begin micro-behaviour

SET-MAXIMUM-LIFE

Begin NetLogo code: 
substitute-text-area-for minimum-lifespan 80
substitute-text-area-for maximum-lifespan 100 
set my-maximum-life random-number-between minimum-lifespan maximum-lifespan
End NetLogo code

Variants

You can use a constant or a different distribution of values for my-maximum-life.

Related Micro-behaviours

DIE-WHEN-TOO-OLD uses this.

How this works

It picks a random number between 80 and 100 and saves it in my-maximum-life.
