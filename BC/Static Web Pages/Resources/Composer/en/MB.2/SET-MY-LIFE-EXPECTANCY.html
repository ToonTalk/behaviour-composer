Begin NetLogo code: 
substitute-text-area-for lifespan-expression random-normal the-average-life-expectancy (the-average-life-expectancy * 0.25)

    time +
    lifespan-expression]
End NetLogo code

Variants

Various distributions can be used but should maintain the the-average-life-expectancy.

Related Micro-behaviours

This micro-behaviour sets my-life-expectancy which is used by DIE-WHEN-LIFE-EXPECTANCY-EXCEEDED. INITIALISE-LIFE-EXPECTANCY is used to create an initial population with a distribution of life expectancies.

How this works

I set my-life-expectancy to the current time plus a random number normally distributed around the-average-life-expectancy with a standard deviation of 1/4 of the-average-life-expectancy.
