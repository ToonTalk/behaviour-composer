I reduce my hunger and the one I'm eating dies. Notice that killing is implemented by giving the other the DIE behaviour.

Begin micro-behaviour

KILL

Begin NetLogo code: 
add-behaviours-to the-other list-of-micro-behaviours "Behaviours when killed" [DIE.html]
add-behaviours-to-me list-of-micro-behaviours "Behaviours when killing" [REDUCE-HUNGER.html]
End NetLogo code

Variants

The REDUCE-HUNGER behaviour is here for illustrative purposes. You can define other things that will happen when killing.

Related Micro-behaviours

The agent that is given the DIE behaviour is terminated.

How this works

This adds the DIE behaviour to the other agent (e.g. the one I'm interacting with). And adds the REDUCE-HUNGER behaviour to myself.  It uses the-other to refer to another agent.
