I set up and maintain a plot of average utility gains of all players versus elapsed time.

Begin micro-behaviour

CREATE-AVERAGE-UTILITY-GAINS-PLOT

Begin NetLogo code: 
create-plot
  "5" "550" ; lower left corner
  "420" "809" ; upper right corner
  "Average Utility Gains Plot" ; unique name (and label) for this plot
  "time" ; X axis label
  "utility gains" ; Y axis label
  time ; X axis value
  average-utility-gains ; Y axis value
End NetLogo code

Variants

You could produce a plot of the average change in utility instead.

Related Micro-behaviours

This uses AVERAGE-UTILITY-GAINS to compute the average of my-accumulated-add-link-utility-gains and my-accumulated-remove-link-utility-gains of all players.
