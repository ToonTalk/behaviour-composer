I create a slider controlling the fraction of my acquaintances that I encounter per time period.

Begin micro-behaviour

CREATE-ENCOUNTER-FRACTION-SLIDER

Begin NetLogo code: 
substitute-text-area-for slider-variable-name the-encounter-fraction               
substitute-text-area-for slider-lower-left-corner-x 625 
substitute-text-area-for slider-lower-left-corner-y 10  
substitute-text-area-for slider-upper-right-corner-x 875 
substitute-text-area-for slider-upper-right-corner-y 50  
substitute-text-area-for slider-minimum-value 0   
substitute-text-area-for slider-maximum-value 1   
substitute-text-area-for slider-increment .01 
substitute-text-area-for slider-initial-value .5  

create-slider 
  "slider-variable-name" ; the name of the parameter (a global variable)
  "slider-lower-left-corner-x" "slider-lower-left-corner-y" ; lower left corner 
  "slider-upper-right-corner-x" "slider-upper-right-corner-y" ; upper right corner 
  "slider-minimum-value" "slider-maximum-value" ; minimum and maximum value
  "slider-increment" ; increment 
  "slider-initial-value" ; initial value
End NetLogo code

Variants

You can change the location, increment, and initial value.

Related Micro-behaviours

CREATE-ENCOUNTER-RATE-SLIDER differs from this slider in that it defines a fixed number of encounters independent of the number of acquaintances. RANDOM-ENCOUNTER uses the parameter the-encounter-rate defined by this slider. Together with CREATE-INFECTION-ODDS-SLIDER this specifies β or the force of infection in [1]. CREATE-DEATH-RATE-SLIDER, CREATE-RATE-OF-RECOVERY-SLIDER, CREATE-FRACTION-VACCINATED-AT-BIRTH-SLIDER and CREATE-RATE-OF-LOSS-OF-IMMUNITY-SLIDER create related sliders.
