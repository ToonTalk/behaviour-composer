When some condition is true I add new micro-behaviours and remove old ones.

Begin micro-behaviour

WHENEVER

Begin NetLogo code: 
substitute-text-area-for condition true                                   
whenever [condition] 
  [add-behaviours    list-of-micro-behaviours "Behaviours to add" []
   remove-behaviours list-of-micro-behaviours "Behaviours to remove" []]
End NetLogo code

Variants

Edit the condition (true) to the desired condition. Add micro-behaviours to the lists of behaviours to add or remove.

Related Micro-behaviours

WHEN runs only once while WHENEVER runs every time the condition is true.

How this works

whenever is a construct added to NetLogo that is checked each time the scheduler advanced the time and every time it is true executes the actions.
