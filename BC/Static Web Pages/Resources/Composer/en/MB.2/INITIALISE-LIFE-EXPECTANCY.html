I set my life expectancy.

Begin micro-behaviour

INITIALISE-LIFE-EXPECTANCY

Begin NetLogo code: 
set my-life-expectancy random-number-between 0 the-average-life-expectancy
End NetLogo code

Variants

One can use other distributions of the initial values for my-life-expectancy.

Related Micro-behaviours

This micro-behaviour is used by CREATE-INITIAL-INFECTED-POPULATION, CREATE-INITIAL-SUSCEPTIBLE-POPULATION and CREATE-INITIAL-VACCINATED-POPULATION]. my-life-expectancy is used by DIE-WHEN-LIFE-EXPECTANCY-EXCEEDED. SET-MY-LIFE-EXPECTANCY is used when someone is born.

How this works

I set my-life-expectancy to a random number uniformly distributed between 0 and the the-average-life-expectancy.
