I jump to a location on a circle.

Begin micro-behaviour

SET-POSITION-ON-A-CIRCLE

Begin NetLogo code: 
substitute-text-area-for radius 8  
let saved-my-heading my-heading
let those-in-the-circle sort all-individuals
set my-heading position self those-in-the-circle * 360 / length those-in-the-circle
set my-next-x 0
set my-next-y 0
go-forward radius
set my-heading saved-my-heading
End NetLogo code

Variants

To change the radius of the circle change the 8 to another number. Change the values my-next-x and my-next-y are set to to centre the circle around a different point. If you want separate circles for different kinds of agents, e.g. "Sheep" then add with [kind = "Sheep"] after all-individuals.

Related Micro-behaviours

SET-RANDOM-POSITION sets my heading to a random value. SET-RANDOM-UNOCCUPIED-POSITION also finds a random position but only if it is unoccupied.

How this works

This temporarily sets my heading to an angle that places each agent 360 divided by the number of agents multiplied by the position of the agent in a list of all agents in the circle. I then move the radius distance in the computed heading. Finally I restore my heading to its original value.
