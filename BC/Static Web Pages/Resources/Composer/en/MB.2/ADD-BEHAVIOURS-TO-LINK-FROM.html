I add behaviours to the link from another to me.

Begin micro-behaviour

ADD-BEHAVIOURS-TO-LINK-FROM

Begin NetLogo code: 
substitute-text-area-for other-end-of-the-link the-other                     
add-behaviours-to-link-from list-of-micro-behaviours "Behaviours" [] other-end-of-the-link
End NetLogo code

Variants

Just add micro-behaviours to the list by clicking on the micro-behaviour and navigating back to this page. You can change the expression that specifies who is at the other end of the link.

Related Micro-behaviours

ADD-BEHAVIOURS-TO-LINK-TO add behaviours to the link to another from me. ADD-BEHAVIOURS adds behaviours to the agent.

Note only some behaviours make sense for a link. For example, SET-COLOR works fine but SET-SHAPE makes no sense.
