Each patch of the world initialises its variables.

Begin micro-behaviour

INITIALISE-EACH-PATCH

Begin NetLogo code: 
substitute-text-area-for patch-command set patches-food-availability 0.0                           
ask-every-patch
    [patch-command]
End NetLogo code

Variants

Any patch variable (begins with patches--) or any built-in NetLogo patch variable (e.g. pcolor) can be set to any value.

Related Micro-behaviours

SET-BACKGROUND-COLOR is an example using this.

How this works

This runs the set command for every patch of the environment.
