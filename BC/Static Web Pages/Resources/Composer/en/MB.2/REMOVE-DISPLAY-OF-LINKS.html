Begin micro-behaviour

REMOVE-DISPLAY-OF-LINKS

Begin NetLogo code: 
do-after .999
   [do-every 1
       [set need-to-clear-drawing true]]
End NetLogo code

Related Micro-behaviours

DISPLAY-ALL-LINKS draws links.

How this works

Just before the current links are displayed this arranges for the erasure of previously drawn links.
