I place a set of agents in a rectangular grid.

Begin micro-behaviour

LAYOUT-GRID

Begin NetLogo code: 
substitute-text-area-for lower-left-x min-pxcor                       
substitute-text-area-for lower-left-y min-pycor                       
substitute-text-area-for grid-width world-width                       
substitute-text-area-for grid-height world-height                                
substitute-text-area-for selection-criteria true                
layout-grid 
  all-individuals with [selection-criteria] 
  lower-left-x  ; lower left x
  lower-left-y  ; lower left y
  grid-width    ; width
  grid-height   ; height
End NetLogo code

Variants

To change the location of the grid of agents change the lower left x and y. To make a grid of a different size edit the width and height. If you want separate rectangles for different kinds of agents, e.g. "Sheep" then replace true with kind = "Sheep" inside the square brackets.

Related Micro-behaviours

LAYOUT-CIRCLE lays out agents in a circle. SET-RANDOM-POSITION sets an agent's heading to a random value. SET-RANDOM-UNOCCUPIED-POSITION also finds a random position but only if it is unoccupied.

How this works

This uses the Modelling4All NetLogo layout-grid command to place the agents in a rectangle.
