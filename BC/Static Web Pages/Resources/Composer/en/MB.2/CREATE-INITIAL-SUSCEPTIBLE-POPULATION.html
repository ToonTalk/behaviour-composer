Related Micro-behaviours

CREATE-INITIAL-SUSCEPTIBLE-COUNT-SLIDER creates a slider for controlling the-initial-susceptible-population. This micro-behaviour relies upon SET-RANDOM-POSITION to initialise the location and SET-STATE-SUSCEPTIBLE to set the state of the copies to "susceptible".

CREATE-INITIAL-INFECTED-POPULATION and CREATE-INITIAL-VACCINATED-POPULATION initialise other parts of the population.

How this works

This creates the-initial-susceptible-population minus one copies of me that also run the SET-STATE-SUSCEPTIBLE and SET-RANDOM-POSITION micro-behaviours. One is subtracted since the initial prototypical person is susceptible.
