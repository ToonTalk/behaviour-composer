I set my state to infected after a tiny delay.

Begin micro-behaviour

SET-STATE-INFECTED-DELAYED

Begin NetLogo code: 
do-after .000003 
  [add-behaviour SET-STATE-INFECTED]
End NetLogo code

How this works

This sets the state to infected after a sufficient delay that the social network should be completely in place. Using this ensures that the log shows the number of acquaintances of those infected.
