I calculate the encounter rate.

Begin micro-behaviour

REPORT-ENCOUNTER-RATE

Begin NetLogo code: 
to-report encounter-rate
 ; a hack to avoid compile time errors since only one of the-encounter-rate and the-encounter-fraction should be defined
 set temp 0
 carefully [run "set temp the-encounter-rate"]
           [carefully [run "set temp the-encounter-fraction * count my-acquaintances with [my-state != \"dead\"]"]
                      [user-message "To compute the encounter rate we need either CREATE-ENCOUNTER-RATE-SLIDER or CREATE-ENCOUNTER-FRACTION-SLIDER together with a social network micro-behaviour."]]
 report temp
end
End NetLogo code

Related Micro-Behaviours

CREATE-ENCOUNTER-RATE-SLIDER or CREATE-ENCOUNTER-FRACTION-SLIDER must be defined for this to work.

How this works

This adds to your model a new NetLogo reporter that can be used by any micro-behaviour. If the-encounter-rate is defined it reports its value. Otherwise if the-encounter-fraction is defined it reports the product of its value and the number of living acquaintances.
