I die with some probability.

Begin micro-behaviour

DEATH-RATE

Begin NetLogo code: 
substitute-text-area-for cycle-duration 1       
do-every cycle-duration
   [do-with-probability reciprocal the-average-life-expectancy
       [add-behaviours list-of-micro-behaviours "Death Behaviours" [SET-STATE-DEAD.html]]]
End NetLogo code

Related Micro-behaviours

CREATE-DEATH-RATE-SLIDER sets the-average-life-expectancy. An alternative way to model death is DIE-WHEN-LIFE-EXPECTANCY-EXCEEDED which uses a pre-computed life expectancy. BIRTH-RATE works in similar manner to this micro-behaviour.

How this works

This computes the odds of dying in any week and uses it to determine whether I die. I die by adding the SET-STATE-DEAD behaviour.
