I create a slider the maximum distance for my local interactions.

Begin micro-behaviour

CREATE-SLIDER-FOR-LOCAL-INTERACTION

Begin NetLogo code: 
substitute-text-area-for slider-variable-name the-local-interaction               
substitute-text-area-for slider-lower-left-corner-x 625 
substitute-text-area-for slider-lower-left-corner-y 10  
substitute-text-area-for slider-upper-right-corner-x 875 
substitute-text-area-for slider-upper-right-corner-y 50  
substitute-text-area-for slider-minimum-value 0   
substitute-text-area-for slider-maximum-value 20 
substitute-text-area-for slider-increment .1   
substitute-text-area-for slider-initial-value 6  

create-slider 
  "slider-variable-name" ; the name of the parameter (a global variable)
  "slider-lower-left-corner-x" "slider-lower-left-corner-y" ; lower left corner 
  "slider-upper-right-corner-x" "slider-upper-right-corner-y" ; upper right corner 
  "slider-minimum-value" "slider-maximum-value" ; minimum and maximum value
  "slider-increment" ; increment 
  "slider-initial-value" ; initial value
End NetLogo code

Variants

You can change the location, and the initial, minimum and maximum values of this slider.

Related Micro-behaviours

CREATE-SLIDER-FOR-PERSONAL-SPACE creates slider for the maximum personal space distance. DIRECTION-TO-ALIGN-WITH-OTHERS and DIRECTION-TOWARDS-OTHERS uses the variable defined by this slider.

How this works

This creates a NetLogo slider for the the-personal-space variable.
