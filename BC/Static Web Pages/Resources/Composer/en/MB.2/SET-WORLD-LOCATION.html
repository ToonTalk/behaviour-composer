I change the location of the display of the world.

Begin micro-behaviour

SET-WORLD-LOCATION

Begin NetLogo code: 
substitute-text-area-for lower-left-world-x 200 
substitute-text-area-for lower-left-world-y 10
substitute-text-area-for upper-right-world-x 650 
substitute-text-area-for upper-right-world-y 550
set-world-location lower-left-world-x lower-left-world-y upper-right-world-x upper-right-world-y
End NetLogo code

Variants

set-world-location expects 4 numbers: lower left corner x, lower left corner y, upper right corner x, and upper right corner y. This example creates a 100x100 grid since -49 to 50 inclusive is 100 integers including 0.

How this works

Due to current limitations of NetLogo this specifies only the initial location of the world display and cannot be used to change the world location during execution.
