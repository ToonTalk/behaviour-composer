This delays the execution of a list of micro-behaviours for a link.

Begin micro-behaviour

LINK-DO-AFTER

Begin NetLogo code: 
substitute-text-area-for delay-duration 1      
add-link-behaviours-after delay-duration 
    list-of-micro-behaviours "Delayed Link Behaviours" []
End NetLogo code

Related Micro-behaviours

DO-AFTER delays the execution for an agent.

Variants

You can edit the duration field to any positive number. You can add micro-behaviours to the list.

How this works

This relies upon the Behaviour Composer scheduling extensions to NetLogo to run the list of micro-behaviours of a link after a delay.

History
