I create a slider controlling the duration of vaccination.

Begin micro-behaviour

CREATE-RATE-OF-LOSS-OF-IMMUNITY-SLIDER

Begin NetLogo code: 
create-slider 
  "the-average-duration-of-vaccination" 
  "625" "10" ; lower left corner 
  "875" "50" ; upper right corner
  "0" "5200" ; minimum and maximum value
  "52" ; increment 
  "1040" ; initial value
End NetLogo code

Variants

You can change the location, minimum and maximum values, increment, and initial value. The units are in weeks.

Related Micro-behaviours

POSSIBLY-LOSE-IMMUNITY uses the parameter defined by this slider. CREATE-INFECTION-ODDS-SLIDER, CREATE-DEATH-RATE-SLIDER, CREATE-RATE-OF-RECOVERY-SLIDER and CREATE-FRACTION-VACCINATED-AT-BIRTH-SLIDER create related sliders.
