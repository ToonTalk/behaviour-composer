I set my age to zero.

Begin micro-behaviour

AGE-IS-ZERO

Begin NetLogo code: 
do-now
   [set my-birth time]
End NetLogo code

Since "age" is computed relative to birth this records the time of birth.

Variants

You can initialise or update any attribute to any value (a number, string, list, other object, or set of objects).

Related Micro-behaviours

INITIALISE-ATTRIBUTE is similar to this behaviour. GROW-OLDER increases the age. SIZE-IS-FUNCTION-OF-AGE uses age to determine the size.

History

This was implemented by Ken Kahn. This relies upon features introduced with the 18 December 2006 BehaviourComposer release.
