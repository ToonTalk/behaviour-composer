I create a slider controlling the fraction of my acquaintances that I encounter per time period.

Begin micro-behaviour

CREATE-ENCOUNTER-FRACTION-SLIDER

Begin NetLogo code: 
create-slider 
  "the-encounter-fraction" 
  "625" "10" ; lower left corner 
  "875" "50" ; upper right corner
  "0" "1" ; minimum and maximum value
  ".01" ; increment 
  ".5" ; initial value
End NetLogo code

Variants

You can change the location, increment, and initial value.

Related Micro-behaviours

CREATE-ENCOUNTER-RATE-SLIDER differs from this slider in that it defines a fixed number of encounters independent of the number of acquaintances. RANDOM-ENCOUNTER uses the parameter the-encounter-rate defined by this slider. Together with CREATE-INFECTION-ODDS-SLIDER this specifies β or the force of infection in [1]. CREATE-DEATH-RATE-SLIDER, CREATE-RATE-OF-RECOVERY-SLIDER, CREATE-FRACTION-VACCINATED-AT-BIRTH-SLIDER and CREATE-RATE-OF-LOSS-OF-IMMUNITY-SLIDER create related sliders.
