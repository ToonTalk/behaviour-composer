Each patch of the world updates the availability of food repeatedly.

Begin micro-behaviour

EACH-PATCH-CHANGES

Begin NetLogo code: 
do-every delta-t
   [ask-every-patch
        [set patches-food-availability
             patches-food-availability +
             random-number-between 0 patches-maximum-food-production]]
End NetLogo code

How this works

This repeatedly asks every patch to increment its patches-food-availability to by a random number between 0 and patches-maximum-food-production.
