I initialise myself by initialising my sugar level, sugar consumption, vision level, age, and life expectancy.

Begin micro-behaviour

INITIALISE-AGENT

Begin NetLogo code: 
add-behaviours [INITIALISE-SUGAR-CONSUMPTION
	        CREATE-SLIDER-MINIMUM-SUGAR-CONSUMPTION
                CREATE-SLIDER-MAXIMUM-SUGAR-CONSUMPTION
                INITIALISE-VISION-LEVEL
                CREATE-SLIDER-MINIMUM-VISION-LEVEL
                CREATE-SLIDER-MAXIMUM-VISION-LEVEL
                INITIALISE-SUGAR-ENDOWMENT
                INITIALISE-AGE
                INITIALISE-LIFETIME]
End NetLogo code

Variants

If the attributes initialised by these micro-behaviours are not used then the micro-behaviours can be removed from the list. Other micro-behaviours can be added to the list.

Related Micro-behaviours

CREATE-NEW-AGENT and CREATE-INITIAL-POPULATION use this micro-behaviour. INITIALISE-SUGAR-CONSUMPTION, INITIALISE-VISION-LEVEL, INITIALISE-SUGAR-ENDOWMENT, INITIALISE-AGE, and INITIALISE-LIFETIME are added by this micro-behaviour. It also adds sliders controlling the range of values chosen within INITIALISE-SUGAR-CONSUMPTION and INITIALISE-VISION-LEVEL.
