I remove a link to another.

Begin micro-behaviour

REMOVE-LINK

Begin NetLogo code: 
if link-with the-other != nobody [ask link-with the-other [die]]
End NetLogo code

Related Micro-behaviours

ADD-DIRECTED-LINK creates a directed link. ADD-UNDIRECTED-LINK creates an undirected link.

How this works

This uses the NetLogo link-with primitive with the value the-other to get the link and then removes it using the NetLogo die command.
