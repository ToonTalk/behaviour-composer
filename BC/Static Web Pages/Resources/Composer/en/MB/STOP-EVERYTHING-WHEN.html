I pause the execution of the simulation when some condition is true.

Begin micro-behaviour

PAUSE-WHEN-TIME-EXCEEDS-50

Begin NetLogo code: 
when [time > ]
     [set stop-running true
      output-print "Paused because time exceeds maximum."]
End NetLogo code

Variants

You can edit the number in the text box to any positive value. You can specify a different condition by replacing time > 50 with some other condition.

How this works

When the first condition (time > 50) is true it sets a flag telling the program to pause at the end of this step.
