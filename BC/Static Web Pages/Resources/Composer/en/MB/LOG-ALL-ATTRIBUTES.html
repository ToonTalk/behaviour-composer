Whenever one of my attributes changes, the event is logged in the output window.

Begin micro-behaviour

LOG-ALL-ATTRIBUTES

Begin NetLogo code: 
do-now 
   [set log-attributes true]
End NetLogo code

Variants

You can change the size and location of the output area by adding the following after "true".

   ; and create log area (lower left corner followed by upper right)
   create-output-area 624 10 1246 546

Related Micro-behaviours

LOG-SPECIFIED-ATTRIBUTES is an example of selectively logging attributes. You can also log behaviours using LOG-ALL-BEHAVIOURS and LOG-SPECIFIED-BEHAVIOURS.
