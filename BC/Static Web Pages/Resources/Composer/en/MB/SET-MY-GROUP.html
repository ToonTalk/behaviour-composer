I set the group I'm in.

Begin micro-behaviour

SET-MY-GROUP

Begin NetLogo code: 
do-at-setup
   [set my-group random-integer-between 1 3]
End NetLogo code

Variants

Change the range or the distribution of group identifiers.

Related Micro-behaviours

TOTAL-COST uses my-group.

How this works

This sets the value of my-group for each player to 1, 2, or 3.

History

This was implemented by Ken Kahn.
