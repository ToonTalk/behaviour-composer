I set up and maintain a plot of my hunger versus elapsed time.

Begin micro-behaviour

CREATE-PLOT

Begin NetLogo code: 
create-plot
  "5" "550" ; lower left corner
  "420" "809" ; upper right corner
  "Hunger Plot" ; unique name (and label) for this plot
  "time" ; X axis label
  "hunger" ; Y axis label
  time ; X axis value
  my-hunger ; Y axis value
  "0" ; minimum x value
  "100" ; maximum x value
  "0"  ; minimum y value
  "200" ; maximum y value
End NetLogo code

Variants

One can plot other attributes. You need not have time as the x-axis. This code updates the graph named "Hunger Plot".

You don't need to specify the location of the plot if the default is acceptable. Just remove those lines. Also you needn't specify the X and Y values since they can be handled by other code. See, for example, PLOT-MY-OPINION.

You can also remove the last four arguments that specify the minimum and maximum values. The NetLogo autoplot feature will rescale the plot as needed.

You can replace my-hunger with other attributes. For example, use 

count all-individuals with [kind = "fish"]

instead of "my-hunger" to graph the fish population.

Note that currently the system is limited to interpreting calls to create-plot where each argument is on a different line.

Related Micro-behaviours

CREATE-OPINION-PLOT and PLOT-AVERAGE-DIRECTION are two other examples of creating plots. CREATE-HISTOGRAM creates histograms.
