I grow older.

Begin micro-behaviour

AGE

Begin NetLogo code: 
do-every delta-t
   [set my-age my-age + delta-t]
End NetLogo code

Related Micro-behaviours

DIE-IF-TOO-OLD uses the my-age attribute.

How this works

Every delta-t it updates my-age by delta-t. Note that my-age doesn't need to be initialised since attributes are automatically initialised to zero.
