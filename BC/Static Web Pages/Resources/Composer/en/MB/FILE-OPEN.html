I open a file. I create it if doesn't exist. I add to it if already does.

Begin micro-behaviour

FILE-OPEN

Begin NetLogo code: 
do-at-setup

 [file-open "sample.txt"]
End NetLogo code

Variants

You can use any file name.

Related Micro-behaviours

FILE-READ, FILE-TYPE, FILE-WRITE, and FILE-CLOSE can be used to read, write, or close a file.

How this works

It calls the NetLogo file-open primitive.
