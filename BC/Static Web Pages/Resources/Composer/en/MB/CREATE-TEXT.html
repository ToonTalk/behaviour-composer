I add some text to the NetLogo interface.

Begin micro-behaviour

CREATE-TEXT

Begin NetLogo code: 
create-text
  "5" "250" ; lower left corner 
  "80" "300" ; upper right corner
  "line 1" ; first line of text
  "line 2" ; second line
  "final line" ; third line
End NetLogo code

Variants

All the parameters can be changed. You can enter as many lines of text as you wish. It can also be edited within NetLogo but those changes won't be saved.

Related Micro-behaviours

CREATE-PLOT, CREATE-SLIDER, CREATE-MONITOR, CREATE-CHOOSER and CREATE-LEFT-TURN-BUTTON also create interface elements.

How this works

This creates a NetLogo chooser for a global variable. In this example the variable is named a-b-or-c and the choices are "a", "b", or "c".
