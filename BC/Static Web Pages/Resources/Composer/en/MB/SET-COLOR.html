I change my colour.

Begin micro-behaviour

SET-COLOR

Begin NetLogo code: 
set color
End NetLogo code

Variants

Named colours are black, white, gray, red, orange, brown, yellow, green, lime, turquoise, cyan, sky, blue, violet, magenta, and pink. More colors can be specified.

To pick a random colour replace the contents of the edit box with

(list random-number-between 0 255   ; red
      random-integer-between 0 255  ; green
      random-integer-between 0 255) ; blue

Related Micro-behaviours

SET-SHAPE is for changing the shape of an object.

How this works

This sets the NetLogo primitive attribute color to the NetLogo defined colour named red.
