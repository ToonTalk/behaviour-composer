I create copies of myself at setup. In the sample code below, each copy has the additional behaviour of jumping to a random position. The newly created copies can be given any additional behaviours (here the SET-POSITION-ON-A-CIRCLE and INITIALISE-MY-LINKS behaviours were added).

Begin micro-behaviour

ADD-PLAYERS

Begin NetLogo code:  

do-at-setup
   [add-copies (the-total-number-of-players - 1) [SET-POSITION-ON-A-CIRCLE INITIALISE-MY-LINKS SET-MY-GROUP]]
End NetLogo code

Related Micro-behaviours

This uses INITIALISE-MY-LINKS to initialise the links of the newly created copies. SET-POSITION-ON-A-CIRCLE arranges the players in a circle. TOTAL-NUMBER-OF-PLAYERS-SLIDER sets the value of total-number-of-players.
