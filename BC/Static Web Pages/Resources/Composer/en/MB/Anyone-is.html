Anyone-is

The primitive anyone-is takes one argument in square brackets. That argument should be a predicate that when run returns true or false. Note if it returns true it also sets the-other to an object that satisfies the predicate. For example, 

when
 anyone-is [size > 1]
 [add-behaviour-to the-other <url to micro-behaviour>]
Anyone-who-is is an alternative name for this NetLogo extension.
