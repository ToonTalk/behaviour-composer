I report my cost of my links.

Begin micro-behaviour

TOTAL-COST

Begin NetLogo code: 
to-report total-cost [player-links]
   if empty? player-links [report 0]
   ifElse uninitialised? my-group
     [report the-direct-link-cost * length player-links]
     [report reduce [?1 + ?2]
                    map [cost-of-link my-group [my-group] of ?] player-links]
end

to-report cost-of-link [group1 group2]
   ifElse group1 = group2
     [report the-same-group-direct-link-cost]
     [report the-direct-link-cost]
end
End NetLogo code

Variants

Other cost functions can replace this one.

Related Micro-behaviours

This is used by UTILITY-OF-NETWORK. SET-MY-GROUP defines my-group. DIRECT-LINK-SAME-GROUP-COST-SLIDER and DIRECT-LINK-BENEFIT-SLIDER define the parameters of this function. ADD-NETWORK-FORMATION-PROCEDURES adds this reporter to the program.

How this works

If my-group isn't defined then every link costs the-direct-link-cost. Otherwise the sum of costs of each direct link is added where same group links have a different cost (typically lower) than different groups.
