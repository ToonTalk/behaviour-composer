I increase my size with age until I reach full size.

Begin micro-behaviour

SIZE-IS-FUNCTION-OF-AGE

Begin NetLogo code: 
do-every delta-t
   [set size minimum (.5 + 0.01 * my-age) 3]
End NetLogo code

Variants

You can change the initial size (.5 in this example) or the maximum size (3 here). Or the growth rate (0.01 here). Size may be a more complex function of age (or other factors).

Related Micro-behaviours

GROW-OLDER increases the age of an object.

How this works

After every delta-t time units this sets my size to a function of my-age with a maximum value of 3.
