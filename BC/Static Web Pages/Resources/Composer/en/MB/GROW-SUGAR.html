The environment grows sugar on each patch until its capacity is reached.

Begin micro-behaviour

GROW-SUGAR

Begin NetLogo code: 
do-every delta-t
   [ask-every-patch
        [let previous-sugar sugar-of-patch 
         set sugar-of-patch minimum (sugar-of-patch + (sugar-growth-rate-of-patch * delta-t))
                                    sugar-capacity-of-patch
         set sugar-production-of-patch sugar-of-patch - previous-sugar]]
End NetLogo code

Related Micro-behaviours

INITIALISE-SUGAR-CAPACITY and INITIALISE-SUGAR-GROWTH-RATE initialise the parameters used here. GROW-SUGAR-DELAYED should be used instead since it delays this behaviour until after the sugar has been collected by agents.

How this works

It adds the product of sugar-growth-rate-of-patch and delta-t to the sugar-of-patch of each patch until it reaches the maximum sugar-capacity-of-patch.
