I initialise the uncertainty with which I hold an opinion to a low value (i.e. I'm pretty certain).

Begin micro-behaviour

SET-CERTAIN-OPINION

Begin NetLogo code: 
do-at-setup
   [set my-uncertainty 0.1]
End NetLogo code

Variants

You can a different uncertainty value instead of 0.1, use a random number between 0 and 1 instead, or the value of a slider CREATE-SLIDER.

Related Micro-behaviours

The my-uncertainty attribute is used by RELATIVE-AGREEMENT. SET-OPINION-UNCERTAINTY sets it to a more normal value. SET-RANDOM-OPINION should also be used to set the "level" of this opinion.

How this works

It sets the attribute my-uncertainty to a 0.1 as was done for the extremists in the model described in [1].

History

This was implemented by Ken Kahn.

Footnotes
