If you do change the shape name, we recommend you also change the behaviour name and the introductory text to avoid confusion.

How this works

This sets the NetLogo primitive attribute called shape to the name of a shape.

Related Micro-behaviours

SET-SIZE and SET-COLOR also change the appearance of objects.
