I initialise my opinion to a random value.

Begin micro-behaviour

SET-RANDOM-OPINION

Begin NetLogo code: 
do-now
   [set my-opinion random-number-between -1 1]
End NetLogo code

Variants

You can use a different distribution (this one is uniform between -1 and 1).

Related Micro-behaviours

This attribute is used by RELATIVE-AGREEMENT. SET-OPINION-UNCERTAINTY should also be used to set the uncertainty of this opinion. SET-EXTREME-OPINION is for setting extreme and relatively certain opinions.

How this works

It sets the attribute my-opinion to a uniform random value between -1 and 1. It is part of an implementation of the model in [1].

History

This was implemented by Ken Kahn.

[1] An implementation of "How can extremism prevail? A study based on the relative agreement interaction model" by Guillaume Deffuant, Frédéric Amblard, Gérard Weisbuch and Thierry Faure (2002); Journal of Artificial Societies and Social Simulation vol. 5, no. 4; <http://jasss.soc.surrey.ac.uk/5/4/1.html#2.3>
