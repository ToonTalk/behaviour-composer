I draw a line to all my acquaintances.

Begin micro-behaviour

DISPLAY-LINE-TO-MY-ACQUAINTANCES

Begin NetLogo code: 
do-after .01
   [let me self
    ask my-acquaintances [draw-line self me yellow]]
End NetLogo code

Variants

You can change yellow to another color.

Related Micro-behaviours

DISPLAY-LINE-OF-INFECTION displays a line between the agent and recipient of an infection.

SET-SIZE-PROPORTIONAL-TO-NUMBER-OF-MY-ACQUAINTANCES sets the size of the display of individuals to be proportional to the number of acquaintances.

How this works

I ask each agent that is in my-acquaintances to draw a line back to me.
