My hunger decreases by 50 units but can never get below 0. Designed to be used as part of an eating behaviour.

Begin micro-behaviour

REDUCE-HUNGER

Begin NetLogo code: 
set my-next-hunger maximum 0 (my-next-hunger - 50)
End NetLogo code

Variants

An attribute can be set to a value that is a function of its previous value or any other attributes.
