I possibly lose my vaccination and become susceptible.

Begin micro-behaviour

POSSIBLY-LOSE-IMMUNITY

Begin NetLogo code: 
do-every 1
   [do-with-probability reciprocal the-average-duration-of-vaccination
       [add-behaviour SET-STATE-SUSCEPTIBLE
        remove-behaviour POSSIBLY-LOSE-IMMUNITY]]
End NetLogo code

Related Micro-behaviours

POSSIBLY-RECOVER works in a similar manner.

How this works

Every time unit (i.e. week), with probability of 1 / the-average-duration-of-vaccination this resets my state to "susceptible". Once susceptible I no longer need to run this micro-behaviour anymore.
