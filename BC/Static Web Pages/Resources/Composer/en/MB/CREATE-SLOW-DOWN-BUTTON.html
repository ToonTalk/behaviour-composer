I create a button for slowing down.

Begin micro-behaviour

CREATE-SLOW-DOWN-BUTTON

Begin NetLogo code: 
create-button 
  "5" "225" ; lower left corner 
  "107" "255" ; upper right corner
  "SLOW DOWN" ; label
  "set my-next-speed my-next-speed - 0.1" ; action
  "S" ; keyboard shortcut
End NetLogo code

This assumes a micro-behaviour such as GO-FORWARD-AT-SPEED is also used.

Variants

All the parameters can be changed. If you don't want a keyboard shortcut then remove the last line of code. Buttons can also be edited within NetLogo but those changes won't be saved.

Related Micro-behaviours

CREATE-SPEED-UP-BUTTON increase my speed. CREATE-RIGHT-TURN-BUTTON and CREATE-LEFT-TURN-BUTTON are for turning.

CREATE-PLOT and CREATE-SLIDER also create an interface element.

How this works

This creates a NetLogo button. The last parameter, the action, is available to micro-behaviours such as DO-BUTTON-COMMAND as the value of the button-command variable.
