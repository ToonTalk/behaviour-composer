I create copies of myself at setup with random opinions.

Begin micro-behaviour

ADD-RANDOM-POPULATION

Begin NetLogo code: 
do-at-setup
   [add-copies  [SET-RANDOM-POSITION SET-RANDOM-OPINION]]
End NetLogo code

Variants

Replace the contents of the edit box by any positive integer to make a different number of copies.

Related Micro-behaviours

This is intended to be used together with other micro-behaviours for modelling relative agreement.

History

This was implemented by Ken Kahn in March 2009.
