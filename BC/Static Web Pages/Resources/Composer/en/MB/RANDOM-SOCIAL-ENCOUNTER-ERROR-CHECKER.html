This checks at run-time that other micro-behaviours that RANDOM-SOCIAL-ENCOUNTER depends upon have been added before running.

Begin micro-behaviour

RANDOM-SOCIAL-ENCOUNTER-ERROR-CHECKER

Begin NetLogo code: 
do-now
  [if uninitialised? my-acquaintances
      [user-message (word "RANDOM-SOCIAL-ENCOUNTER requires the network be initialised by"
                          " INITIALISE-SOCIAL-NETWORK-CONSTANT-SYMMETRIC,"
                          " INITIALISE-SOCIAL-NETWORK-NORMAL-DISTRIBUTION-SYMMETRIC,"
                          " or INITIALISE-SOCIAL-NETWORK-POWER-LAW-DISTRIBUTION-SYMMETRIC")
;      remove-behaviour RANDOM-SOCIAL-ENCOUNTER
]]
End NetLogo code

Related Micro-behaviours

This is added automatically by RANDOM-SOCIAL-ENCOUNTER.

How this works

Before the RANDOM-SOCIAL-ENCOUNTER behaviour runs this checks that some other micro-behaviour has initialised my-acquaintances.
