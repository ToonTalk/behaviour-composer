I initialise the distance along grid lines that I can see the amount of sugar on a patch.

Begin micro-behaviour

INITIALISE-VISION-LEVEL

Begin NetLogo code: 
set my-vision-level random-integer-between the-minimum-vision-level the-maximum-vision-level
End NetLogo code

Variants

Vision level need not be selected uniformly from the interval [the-minimum-vision-level, the-maximum-vision-level] but can be real numbers or have other distributions.

Related Micro-behaviours

MOVE-TO-NEAREST-PREFERRED-VISIBLE-PATCH uses this to scan north/south and east/west a distance of my-vision-level to select the best patch to move to. INITIALISE-AGENT adds this micro-behaviour.

How this works

This sets my-vision-level to a random integer between the-minimum-vision-level and the-maximum-vision-level inclusive.
