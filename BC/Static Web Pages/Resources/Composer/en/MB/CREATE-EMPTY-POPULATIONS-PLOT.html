I set up and label a plot that other micro-behaviours can draw upon.

Begin micro-behaviour

CREATE-EMPTY-POPULATIONS-PLOT

Begin NetLogo code: 
create-plot
  "5" "550" ; lower left corner 
  "605" "750" ; upper right corner
  "Populations" ; unique name (and label) for this plot
  "time" ; X axis label
  "people" ; Y axis label
End NetLogo code

Variants

All the labels can be changed.

Note that currently the system is limited to interpreting calls to start-plot where each argument is on a different line.

Related Micro-behaviours

ADD-SUSCEPTIBLE-TO-POPULATIONS-PLOT, ADD-VACCINATED-TO-POPULATIONS-PLOT, ADD-INFECTED-TO-POPULATIONS-PLOT, ADD-RECOVERED-TO-POPULATIONS-PLOT, and ADD-TOTAL-TO-POPULATIONS-PLOT can add to this plot.
