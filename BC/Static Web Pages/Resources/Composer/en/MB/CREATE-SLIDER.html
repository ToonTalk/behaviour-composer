I create a slider for a parameter.

Begin micro-behaviour

CREATE-SLIDER

Begin NetLogo code: 
create-slider 
  "" ; the name of the parameter (a global variable)
  "" "" ; lower left corner 
  "" "" ; upper right corner 
  "" "" ; minimum and maximum value
  "" ; increment 
  "" ; initial value
End NetLogo code

Variants

All the parameters can be changed. The Behaviour Composer can provide more support if the parameter name begins with "the-". If multiple sliders have the same location they are placed one under another.

Related Micro-behaviours

CREATE-PLOT, CREATE-LEFT-TURN-BUTTON, and CREATE-MONITOR also create interface elements.

How this works

This creates a NetLogo slider.
