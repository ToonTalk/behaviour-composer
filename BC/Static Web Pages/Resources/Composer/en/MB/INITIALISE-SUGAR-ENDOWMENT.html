I set the amount of sugar I start off with.

The Code

Begin micro-behaviour

INITIALISE-SUGAR-ENDOWMENT

Begin NetLogo code: 
set my-sugar random-integer-between 5 25
End NetLogo code

Variants

The initial sugar endowment of newly created agents can be set to other ranges of numbers and other distributions. It need not be restricted to integer values.

Related Micro-behaviours

Used by INITIALISE-AGENT.

How this works

It sets my-sugar to a random integer between 5 and 25 inclusive.
