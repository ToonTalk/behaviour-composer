I create a slider controlling the duration of an infection.

Begin micro-behaviour

CREATE-RATE-OF-RECOVERY-SLIDER

Begin NetLogo code: 
create-slider 
  "the-average-duration-of-infection" 
  "625" "10" ; lower left corner 
  "875" "50" ; upper right corner
  "0" "10" ; minimum and maximum value
  ".1" ; increment 
  "2" ; initial value
End NetLogo code

Variants

You can change the location, minimum and maximum values, increment, and initial value. The units are in weeks.

Related Micro-behaviours

POSSIBLY-RECOVER uses the parameter defined by this slider. CREATE-INFECTION-ODDS-SLIDER, CREATE-FRACTION-VACCINATED-AT-BIRTH-SLIDER and CREATE-RATE-OF-LOSS-OF-IMMUNITY-SLIDER create related sliders.
