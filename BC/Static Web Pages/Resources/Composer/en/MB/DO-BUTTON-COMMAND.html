I do the command associated with a button.

Begin micro-behaviour

DO-BUTTON-COMMAND

Begin NetLogo code: 
do-every delta-t
   [run button-command
    set button-command "" ; so it is only done once
    ]
End NetLogo code

Related Micro-behaviours

A micro-behaviour to create a button such as: CREATE-LEFT-TURN-BUTTON should be used to create buttons for this behaviour to use.

How this works

When a button is pushed it sets the variable button-command to a command to execute.
