I change the initial size in pixels of each patch.

Begin micro-behaviour

SET-PATCH-SIZE

Begin NetLogo code: 
set-patch-size 12.5
End NetLogo code

Variants

set-patch-size expects one number that is the number of pixels per patch. If, for example, the world is 10x20 patches then the above will make the world display be 125x250 pixels.

How this works

Due to current limitations of NetLogo this specifies only the initial size of each patch and cannot be used to change the patch size during execution.
