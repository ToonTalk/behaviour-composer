I initialise my life expectancy to a random value within a range.

Begin micro-behaviour

INITIALISE-LIFETIME

Begin NetLogo code: 
set my-life-expectancy random-integer-between 60 100
End NetLogo code

Variants

Other numbers and other random distributions are possible.

Related Micro-behaviours

DIE-IF-TOO-OLD causes me to die if my-age exceeds my-life-expectancy.

How this works

It initialises my-life-expectancy to a random integer between 60 and 100 inclusive.

Source

Based on agent replacement rule on page 32 of [1] where a is 60 and b is 100.

History

This was implemented by Ken Kahn.

References

[1] Growing Artificial Societies by Joshua M. Epstein and Robert Axtell, Brookings Institute Press, The MIT Press, 1996.
