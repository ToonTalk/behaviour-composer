I create a monitor for the current simulated time.

Begin micro-behaviour

CREATE-TIME-MONITOR

Begin NetLogo code: 
create-monitor
  "Time" ; label of the monitor
  "5" "350" ; lower left corner
  "105" "400" ; upper right corner
  "time" ; value being monitored
  "0" ; number of decimals
End NetLogo code

Variants

All the parameters can be changed.

Related Micro-behaviours

CREATE-VISION-LEVEL-MONITOR, CREATE-MEAN-METABOLISM-MONITOR, and CREATE-GINI-COEFFICIENT-MONITOR monitor other aspects of the model.
