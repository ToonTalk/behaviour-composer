CREATE-PLOT-OLD-VERSION

Summary

I set up and maintain a plot of my hunger versus elapsed time in start-plot.

The Code

Begin micro-behaviour

CREATE-PLOT

Begin NetLogo code: 
create-plot
  "Hunger Plot" ; unique name (and label) for this plot
  "time" ; X axis label
  real-time ; X axis value
  "hunger" ; Y axis label
  my-hunger ; Y axis value
End NetLogo code

Variants

One can plot other attributes. They need not have time as the x-axis. You can construct multiple plots using the NetLogo interface. This code updates the graph named "Hunger Plot". The interface also enables you to re-label the axes.

You can replace my-hunger with other attributes. For example, use 

count objects with [my-species = "salmon"]

instead of "my-hunger" to graph the salmon population.

Note that currently the system is limited to interpreting calls to start-plot where each argument is on a different line.

Related Micro-behaviours

CREATE-OPINION-PLOT and PLOT-AVERAGE-DIRECTION are two other examples of creating plots.
