I replace myself when dying by creating a new replacement agent.

Begin micro-behaviour

REPLACE-DEAD-AGENT

Begin NetLogo code: 
when [dead]
     [add-copies 1 [INITIALISE-AGENT SET-RANDOM-UNOCCUPIED-POSITION REPLACE-DEAD-AGENT]]
End NetLogo code

Related Micro-behaviours

dead is set by DIE which is run by DIE-IF-NO-SUGAR and DIE-IF-TOO-OLD. This relies upon INITIALISE-AGENT and SET-RANDOM-UNOCCUPIED-POSITION.

Source

Based upon page 32 of [1].

How this works

When dead is TRUE then this rule will be run before I'm removed from the model. It creates a copy of myself except for the effects of INITIALISE-AGENT and SET-RANDOM-UNOCCUPIED-POSITION. The newly created copy also needs this rule and hence is included in the list of micro-behaviours.

History
