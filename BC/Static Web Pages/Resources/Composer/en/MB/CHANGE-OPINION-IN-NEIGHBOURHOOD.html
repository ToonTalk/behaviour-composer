If I get near someone I may change my opinion. This implements part of the model in [1].

Begin micro-behaviour

CHANGE-OPINION-IN-NEIGHBOURHOOD

Begin NetLogo code: 
do-every delta-t     
   [anyone-who-is [1 / (1 + distance-to-me) > random-number-between 0 1]
       [add-behaviour-to self RELATIVE-AGREEMENT
        add-behaviour-to the-other RELATIVE-AGREEMENT]]
End NetLogo code

Variants

You can change the formula used to compute the probability of interacting. If you replace anyone-who-is with all-who-are then each agent will interact will a set agents instead of a single member of the set.

To see a log of the changes in opinion and uncertainty copy and paste the following before "]]": 

 output-print (word "self: " who  " opinion: " my-opinion " next-opinion: " my-next-opinion)
 output-print (word "other: " who-of the-other " uncertainty: " my-uncertainty)

Related Micro-behaviours

This relies upon RELATIVE-AGREEMENT. CHANGE-OPINION-SOCIAL-NETWORK differs from this in that the interaction is with acquaintances rather than someone located spatially nearby.

How this works

On every tick I select someone at random and the probability that I interact with him is inversely proportional to our distance. I interact by running the RELATIVE-AGREEMENT micro-behaviour which may change my opinion and uncertainty.
