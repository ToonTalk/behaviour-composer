I add a link if the other player also wants to add the link.

Begin micro-behaviour

ADD-A-LINK-IF-IN-AGREEMENT

Begin NetLogo code: 
do-after .002
   [do-if my-desire-to-link = true and [my-desire-to-link] of my-other-player = true
       [add-link-to my-other-player
        ask my-other-player [add-link-to myself]
        set my-desire-to-link false
        set my-accumulated-add-link-utility-gains 
            my-accumulated-add-link-utility-gains + my-add-link-utility-gain
        set my-add-links-count my-add-links-count + 1]]
End NetLogo code

Related Micro-behaviours

This uses ADD-LINK-TO. SIMPLE-ADD-LINK-DECISION sets my-desire-to-link.

How this works

If I want to create a link and so does the other player then we form the link.
