If I'm old enough then I reproduce whenever I come close to a conspecific (an organism of the same species) with some probability.

Begin micro-behaviour

REPRODUCE

Begin NetLogo code: 
whenever [random-number-between 0 1 < .333 and
          (time - my-birth) > my-minimum-reproduction-age and
          can-pick-one all-individuals with [distance-to-me < 2 and my-species = [my-species] of myself]]
  [add-behaviours [UPDATE-MINIMUM-REPRODUCTION-AGE ADD-SOME-OFFSPRING]]
End NetLogo code

This code expects objects to have attributes called my-species, my-birth, and minimum-reproduction-age. These can be set by and SET-SPECIES, AGE-IS-ZERO, and UPDATE-MINIMUM-REPRODUCTION-AGE.

Variants

You can change the probability, add other conditions (e.g. age and gender), and change the minimum distance of an encounter from 2. You can also change the ADD-SOME-OFFSPRING micro-behaviour to change the number and properties of the offspring. Or change UPDATE-MINIMUM-REPRODUCTION-AGE to control how soon again this can reproduce.

Related Micro-behaviours

This relies upon the ADD-SOME-OFFSPRING and UPDATE-MINIMUM-REPRODUCTION-AGE micro-behaviours.

How this works

The conditions between "whenever" and "do-now" are evaluated whenever the time changes. In this case it evaluates to true if random number is less than one third, my age is old enough, anyone is close to me that is of my species. If this is true then this objects gets two new behaviours: ADD-SOME-OFFSPRING and UPDATE-MINIMUM-REPRODUCTION-AGE.
