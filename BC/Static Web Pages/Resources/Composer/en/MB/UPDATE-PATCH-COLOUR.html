Each patch of the Sugarscape repeatedly compute their colour depending upon how much sugar has accumulated.

The Code

Begin micro-behaviour

UPDATE-PATCH-COLOUR

Begin NetLogo code: 
do-every delta-t
   [ask-every-patch
        [set pcolor 49 - sugar-of-patch]]
End NetLogo code

Variants

You could use the colour to depict other attributes or use a different colour scheme.

Related Micro-behaviours

GROW-SUGAR-DELAYED is the process that updates the accumulated sugar of each patch.

How this works

In NetLogo's colour scheme yellow is represented by the number 45. Lighter shades of yellow range between 45 and 49. On every delta-t every patch sets it colour to a shade of yellow. This code assumes that accumulated sugar ranges from 0 to 4.
