I set my state to recovered.

The Code

Begin micro-behaviour

SET-STATE-RECOVERED

Begin NetLogo code: 
set my-next-state "recovered"
set color gray
End NetLogo code

Related Micro-behaviours

SET-STATE-SUSCEPTIBLE, SET-STATE-VACCINATED, and SET-STATE-INFECTED also set my-state.

How this works

This creates the variable my-state and sets it to "recovered". It also sets my colour to gray.
