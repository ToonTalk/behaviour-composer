I create a slider controlling the total number of nodes or players.

Begin micro-behaviour

TOTAL-NUMBER-OF-PLAYERS-SLIDER

Begin NetLogo code: 
create-slider 
  "the-total-number-of-players" 
  "625" "10" ; lower left corner 
  "875" "50" ; upper right corner
  "0" "50" ; minimum and maximum value
  "1" ; increment 
  "10" ; initial value
End NetLogo code

Variants

You can change the location, minimum and maximum values, increment, and initial value.
