I create a monitor for the Gini coefficient to continually display the inequality of the wealth (sugar) distribution.

Begin micro-behaviour

CREATE-GINI-COEFFICIENT-MONITOR

Begin NetLogo code: 
create-monitor 
  "Gini Coefficient" ; label of the monitor
  "5" "200" ; lower left corner 
  "105" "250" ; upper right corner
  "[my-gini-coefficient] of any-of-kind 'Sugarscape Environment'" ; value being monitored 
  "3" ; number of decimals
End NetLogo code

Variants

All the parameters can be changed.

Related Micro-behaviours

This requires DRAW-LORENZ-CURVE to compute the Gini coefficient. CREATE-VISION-LEVEL-MONITOR, CREATE-MEAN-METABOLISM-MONITOR and CREATE-TIME-MONITOR monitor other aspects of the model.

How this works

This creates a NetLogo monitor that continually displays the value of [my-gini-coefficient] of any-of-kind 'Sugarscape Environment'. Due to limitations of the NetLogo implementation of monitors the simpler my-gini-coefficient of the Sugarscape entity doesn't work.
