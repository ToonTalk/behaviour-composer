I add to a file.

Begin micro-behaviour

FILE-TYPE

Begin NetLogo code: 
file-type 42
End NetLogo code

Variants

Any value can be added to a file.

Related Micro-behaviours

FILE-OPEN must run first or this will signal an error. FILE-WRITE writes a value in a manner that can be read by FILE-READ.

How this works

It calls the NetLogo file-type.
