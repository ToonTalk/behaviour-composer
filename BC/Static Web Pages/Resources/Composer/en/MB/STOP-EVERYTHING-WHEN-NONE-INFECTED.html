I pause the simulation when no one is infected.

Begin micro-behaviour

PAUSE-WHEN-NONE-INFECTED

Begin NetLogo code: 
when [not any? all-infected and time >= 1]
     [set stop-running true
      output-print (list "Paused because no one is infected. Time is " time)]
End NetLogo code

How this works

When the set of all-infected is empty it sets a flag telling the program to pause at the end of this step.
