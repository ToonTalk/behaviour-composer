MoPiX 2.0

In a related project (ReMath) one of the project members (Ken Kahn) is developing a modelling tool called MoPiX 2.0 with a good deal of overlap with the modelling tool being built in the Modelling4All Project. The current version relies upon equations for modelling but within the Modelling4All Project it is being enhanced to also support NetLogo modelling in the spirit of the BehaviourComposer developed in the JISC-funded Constructing2Learn Project.

Here is the latest version of MoPiX 2.0 (or MoPiX 2.0 customised for Pond Tiling or MoPiX 2.0 for the Sessions on Newtonian Mechanics). (Note that these URLs will resume a previous session if you have used MoPiX earlier in the browser. There will be an interface for starting new sessions but for now use these URLs when you want to start a new session: MoPiX 2.0 - MoPiX 2.0 customised for Pond Tiling - MoPiX 2.0 for the Sessions on Newtonian Mechanics.)

NOTE that the current release sometimes does not work in FireFox. It does work in Internet Explorer, Safari, Opera, and Chrome.
